# 국민연금

## 개요

한국 최대 기관투자자 (자산 1,200조원+). 다수 KOSPI 200 기업에 [[5%-대량보유]] 이상 지분을 보유하며, [[프록시-파이트]]에서 핵심적인 [[의결권]] 행사 주체.

## OPM에서의 확인

- own_block: 국민연금의 대량보유 보고서 (stkrt + rcept_dt). [[DART-OpenAPI]] DS004 majorstock API 사용
- 보유목적: 통상 "단순투자" (경영참여가 아닌 패시브 보유)
- proxy_guideline scope=nps_record: 안건별 국민연금 의결권 행사내역 — fund.nps.or.kr 직접 크롤링 + 정적 JSON 캐싱. 자세한 사양은 [[nps-voting-disclosure]] 참조.
- **정책 schema 통합 완료** (2025.03.01 지침 기준): `open_proxy_mcp/data/asset_managers/policies/nps_2025-03.json`. `_index.json` `nps` 항목에 `policy_file` 등록.
  - 8 자산운용사 schema와 동일 구조 (12 voting_rules 카테고리, general_principles, decision_process 등)
  - **공공기관이라 익명화 X** — `manager_id: "nps"`, `manager_name: "국민연금기금운용본부"` 실명 그대로
  - 별표 1 (국내주식 의결권 행사 세부기준 46항) 기반 추출. 별표 2 (해외주식 50항) 별도 추출 후속 작업

## 정책 핵심

### 의결권 행사 임계치 (지침 제9조)
- 보유지분율 1000분의 10 (1%) 이상 또는
- 국내주식 보유비중 1000분의 5 (0.5%) 이상 (해외 1000분의 3)

### 의사결정 주체 (제5조)
- 기금운용본부 투자위원회 (원칙)
- 수탁자책임 전문위원회 (수책위) — 판단 곤란 사안, 주주가치 영향 큰 사안, 중점관리기업 선정, 주주제안 등

### 단계적 에스컬레이션 (제13~22조)
1. 비공개대화 (1년) — 보유지분율 1000분의 50 (5%) 이상 또는 국내 보유비중 1% 이상 + 중점관리사안 해당 기업
2. 비공개중점관리기업 (선정연도 말 개선 무 또는 비공개대화 거부 시 즉시)
3. 공개중점관리기업 (전문위 결정 시 즉시 가능)
4. 공개서한
5. 주주제안 (기금운용위 결정)
6. 대표소송 (이사 위법행위 + 30일 무대응 시)

### 5대 중점관리사안 ([별표 4])
1. 배당정책 수립
2. 임원 보수한도 적정성
3. 법령위반 우려 (횡령·배임, 부당지원, 사익편취)
4. 지속적 반대표 무개선
6. 기후변화 위험관리 (2023.3.7 신설)
7. 산업안전 위험관리 (2023.3.7 신설)

## 주요 보유 사례

- 삼성전자: 11.52%
- KT&G: 7.50% (최대주주 IBK 8.06%보다 약간 적지만 실질적 영향력 큼)

## 거버넌스 분석에서의 의미

국민연금은 스튜어드십 코드를 통해 의결권 행사 방침을 공개. 주주가치 훼손 안건에 반대표를 던지는 경우가 증가 추세. 5% 이상 보유 기업의 주총에서 실질적 영향력 보유.
