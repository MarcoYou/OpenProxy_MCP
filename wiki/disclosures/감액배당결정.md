# 감액배당결정

자본준비금을 감소시켜 이익잉여금으로 전입한 후, 그 재원으로 배당하는 방식. 이익잉여금이 부족하지만 배당을 원할 때 사용. 주주총회 특별결의 사항(상법 §461). [[정관변경]] 안건과 동반 상정되는 경우가 많음.

## 개요

| 항목 | 값 |
|------|-----|
| 공시 의무 회사 | 자본준비금 감소 결정 회사 |
| 제출처 | DART (주요사항보고서 — DS003 그룹), 주주총회 안건으로 동반 |
| 시기 | 이사회결의일 다음 영업일까지 (주요사항보고서) |
| 의무 여부 | 의무 (자본준비금 감소 시) |
| 공시 주기 | 수시 |
| 한도 | 자본금의 1.5배 초과분만 감소 가능 (상법 §461의2) |
| 결의 | 주주총회 특별결의 (출석 2/3 이상 + 발행주식 1/3 이상) |

## 핵심 데이터 항목

감액배당은 **3단계**로 구성:

### 1단계: 자본준비금 감소 결정 (주요사항보고서)

| 필드 (한글) | 형식 | 예시 | 설명 |
|---|---|---|---|
| 감소할 준비금 종류 | enum | 자본준비금 / 이익준비금 | 통상 자본준비금 |
| 감소 전 준비금 잔액 | int (원) | 5,000,000,000,000 | 감소 전 |
| 감소 금액 | int (원) | 2,000,000,000,000 | |
| 감소 후 준비금 잔액 | int (원) | 3,000,000,000,000 | 감소 후 |
| 자본금 | int (원) | 1,000,000,000,000 | 한도 검증용 |
| 자본금의 1.5배 한도 | int (원) | 1,500,000,000,000 | 상법 §461의2 |
| 전입 대상 | enum | 이익잉여금 | |
| 감소 목적 | text | 배당 재원 확보 | 자유기재 |
| 이사회결의일 | date | 2026-01-29 | |
| 주주총회 예정일 | date | 2026-03-19 | 특별결의 |

### 2단계: 주주총회 안건 (정관변경 + 자본준비금 감소)

- **안건명**: `자본준비금 감소의 件` 또는 `이익잉여금 전입의 件`
- **부의 근거**: 상법 §461 (특별결의)
- **동반 안건**: 정관 변경 (배당가능이익 기준 변경 시), 현금배당 안건

### 3단계: 현금배당 결정 (이익잉여금 확충 후)

→ [[현금배당결정]] 공시 별도 제출

## 전체 문서 구조 (트리, 주요사항보고서)

```
[회사명]/주요사항보고서(자본금감소결정 또는 자본준비금감소결정)/...

자본준비금 감소 결정 (또는 이익잉여금 전입 결정)
│
├── 1. 감소할 준비금
│   ├── 종류            자본준비금
│   ├── 감소 전 잔액    XXX,XXX,XXX
│   ├── 감소 금액       XXX,XXX,XXX
│   └── 감소 후 잔액    XXX,XXX,XXX
├── 2. 자본금
│   ├── 자본금 금액     XXX,XXX,XXX
│   └── 자본금의 1.5배   XXX,XXX,XXX  (한도 비교)
├── 3. 전입 대상         이익잉여금
├── 4. 감소 목적         (자유기재 — 배당 재원 확보 등)
├── 5. 이사회결의일      YYYY-MM-DD
│   ├── 사외이사 참석여부    참석(명) N / 불참(명) N
│   └── 감사 참석여부        -
├── 6. 주주총회 예정일자  YYYY-MM-DD  (특별결의)
├── 7. 채권자 이의제출 절차  (해당 시)
└── 8. 기타 투자판단과 관련한 중요사항
```

## DART API 매핑

- **API**: 별도 구조화 API 없음 (`list.json` + 키워드 검색)
- **report_nm 패턴**:
  - `주요사항보고서(자본준비금감소결정)`
  - `주요사항보고서(자본금감소결정)` (감자와 헷갈림 주의)
  - `[기재정정]` 동반 가능
- **검색 키워드**: `("자본준비금감소", "이익잉여금전입", "감액배당")`
- **DS003 그룹**: 일반 주요사항보고서 (감자결정과 별도)

## OPM 활용

- **현재 미구현 — 신규 tool 또는 기존 tool 확장 필요**
- **기존**: `agm` tool에서 [[정관변경]] 안건 + `capital_reserve` 플래그 (소집공고 본문 파싱)
- **확장 제안**:
  - `dividend(scope=capital_reduction)` — 자본준비금 감소 결정 검색
  - `screen_events(event_type=capital_reduction_for_dividend)` — 시장 전체 검색
- **자본준비금 감소 vs 자본금 감소 ([[감자결정]])** 구분 필수 — 후자는 `dilutive_issuance(scope=capital_reduction)`

## 거버넌스 분석 포인트

- **배당 재원 부족 신호**: 영업현금흐름 음수 + 이익잉여금 부족 → 자본준비금 감액으로 배당 → 지속가능성 의문
- **상법 §461의2 한도 검증**: 자본금의 1.5배 초과분만 감소 가능 — 한도 위반 시 무효
- **채권자 보호 절차**: 이의제출 공고 1개월 + 변제/담보 (자본금 감소와 유사)
- **주주환원 의지 vs 자기 자본 잠식**: 양면적 — 적극 주주환원 OR 부실 신호
- **OPM guideline (default review)**:
  - 영업현금흐름 (+) + 자본준비금 충분 → for
  - 영업현금흐름 (-) + 차입 의존 + 감액배당 → against (과다 배당)

## 신법 영향

- 2026 신법(자기주식 1년 의무소각)과 직접 관계 없음
- 단, **자기주식 처분 → 매각대금으로 배당 재원** 패턴이 차단되면서 감액배당 활용 빈도 증가 가능성
- 2024 자본시장법 개정 + [[배당기준일결정]] 분리 → 감액배당 결정 후 별도 배당기준일 공시 가능
