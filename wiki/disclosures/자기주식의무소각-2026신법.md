# 자기주식 1년 내 의무소각 (2026.03 신법)

2026.03.06 시행된 상법 개정 — **자기주식 취득 후 1년 내 소각 의무화**. 한국 자사주 비중 코스피 시총 7%+ → 글로벌 1-2% 수준 정상화 기대. **자사주 정책 전면 재설계**가 필요한 가장 강력한 신법.

## 개요

| 항목 | 값 |
|------|-----|
| 시행일 | **2026-03-06** (즉시 시행) |
| 근거 법조 | 상법 §341 (취득 사유) + §342 (처분/소각 의무) 개정 |
| 적용 대상 | 모든 상장사 (KOSPI/KOSDAQ/KONEX) |
| 핵심 의무 | 자기주식 취득 후 **1년 내 소각 또는 처분** |
| 기존 보유분 | **6개월 유예 후 1년 내** (2026.03 ~ 2026.09 유예 → 2027.09까지 처리) |
| 위반 효과 | 처분 안건 against 트리거 (OPM guideline) |
| 임팩트 | 한국 자사주 비중 코스피 시총 7%+ → 1-2% 수준 정상화 (글로벌 표준) |

## 신법 내용 (상법 개정 요약)

### 기존 법령 (2026.03 이전)
- 자기주식 취득 가능 (배당가능이익 한도)
- **보유 기간 무제한** — 무기한 보유 가능
- 처분 시 임의 (제3자 매각, 임직원 부여 등 자유)

### 개정 후 (2026.03.06~)
1. **취득 후 1년 내 의무**:
   - 소각 (원칙)
   - 또는 합리적 사유 + 외부 검증 후 처분 (예외)
2. **기존 보유분 처리**:
   - 시행일(2026.03.06) + 6개월 유예 (2026.09.06)
   - 유예 종료 후 1년 내 (2027.09.06)까지 처리 의무
3. **위반 시 효과**:
   - 직접 형사처벌 조항은 없으나
   - **제재**: 거래소 공시 의무 강화 + 감독당국 조치 가능
   - **시장 평가**: ISS/글로벌 기관 against 자동 적용

## 실무 영향

### 자기주식 취득결정 ([[자기주식취득결정]])
| 변경점 | 신법 전 | 신법 후 |
|------|--------|--------|
| `aq_pp` (취득 목적) | "주주가치 제고" 등 모호 가능 | **"소각" 명시 필수** (commitment) |
| 취득 한도 | 배당가능이익 | 동일 + 1년 내 처리 능력 검증 |
| 후속 조치 | 임의 | 1년 내 소각 또는 합리 처분 |

### 자기주식 처분결정 ([[자기주식처분결정]])
| 변경점 | 신법 전 | 신법 후 |
|------|--------|--------|
| 처분 사유 | 자유 (자사주 마법 가능) | **합리적 사유 + 외부 검증** |
| 제3자 매각 | 가능 | 사실상 불가 (against 트리거) |
| 임직원 부여 | 가능 | 가능 (보수 정책의 일부로 인정) |
| 분쟁 중 처분 | 사례 빈번 (한진칼 KCGI) | **against 절대** (정책 강화) |

### 자기주식 소각결정 ([[자기주식소각결정]])
- **준의무화**: 취득 → 1년 내 소각이 원칙
- 빈도 급증 예상 (2026.03 후)
- 이익잉여금 소각이 압도적 다수 (간소함)

### 자기주식 신탁계약 ([[자기주식신탁결정]])
- 체결 단계에서 `ctr_pp`에 "소각" 명시 필수
- 신탁 매입 주식도 1년 내 소각 의무 적용
- 신탁 해지 → 회사 보유 → 1년 카운트다운 시작

## OPM 활용 (treasury_share 재설계)

### 1. 취득 commitment 자동 검증
```python
# services/treasury_share.py _normalize_acquisition()
purpose = item.get("aq_pp", "")
for_cancelation = "소각" in purpose
```
- `for_cancelation=True` → for (commitment 확보)
- `for_cancelation=False` → review/against (1년 후 점검 필요)

### 2. 1년 시점 자동 알람 (신규 기능 제안)
```
취득결정 rcept_dt + 1년 = 점검 시점
점검 시점에 동일 회사의 소각결정 누락 → 워치리스트
```
- `screen_events(event_type="treasury_pending_cancelation")` — 신규 이벤트 타입 제안
- `treasury_share(scope="commitment_check")` — 신규 scope 제안

### 3. 처분 안건 평가 강화
- `dpptncmp_cmpnm` 채워짐 + 합리 사유 미공시 → against 절대
- 분쟁 중 처분 (소송, 5% 보고서 변동, 위임장 권유) → against 절대

### 4. 기존 보유분 추적 (2027.09까지)
- 사업보고서 `tesstkAcqsDspsSttus` (자사주 잔고) — 2026.03 시점 기초 잔고
- 6개월 유예 후 1년 내 처리 모니터링
- 미처리 시 거버넌스 risk 알람

## OPM 의결권 정책 ([[open-proxy-guideline]])

### 2.7 자기주식 (treasury_share) — 2026 신법 전면 재설계

| default | 조건 |
|---------|------|
| **for** | 자기주식 취득 후 1년 내 의무소각 (2026.03 신법 준수) |
| for | 합리적 사유 + 소각 commitment |
| for | 처분 결과 주총 보고 + 처분 주총 승인 의무화 (정관) |
| **against** | 자사주 처분 + 1년 내 소각 계획 없음 (자사주 마법 차단) |
| against | 합리적 사유 없는 새 사업/제3자 배정 처분 |
| **against** | 분쟁 중 자사주 처분 (한진칼 KCGI 패턴) — A3 활동가 |
| review | 경영권 보호 목적 취득 (사실상 against) |

## 글로벌 비교

| 국가 | 자사주 보유 한도/규제 |
|------|------------------|
| **한국 (2026.03 신법)** | **1년 내 소각 의무** (글로벌 최강 수준) |
| 미국 | 무제한 (배당가능이익 한도만) |
| 일본 | 무제한 (단 일정 요건 충족 시 자기주식 수익자 명의) |
| 영국 | 일정 한도 + 의결권 정지 |
| 독일 | 자본금의 10% 한도 |

→ 한국이 **글로벌 최강 의무소각 규제** 시행. 코스피 자사주 비중 7% → 1-2% 수준 정상화 예상.

## 임팩트 추정

### 단기 (2026.03 ~ 2026.09)
- 신법 시행 후 자사주 취득결정 빈도 일시적 감소 (관망)
- 기존 보유분 6개월 유예 — 일부 회사 미리 소각 또는 처분 결정

### 중기 (2026.09 ~ 2027.09)
- 기존 보유분 처리 의무 본격화
- 소각결정 빈도 급증 (연간 50건 → 200건+ 예상)
- 처분결정 빈도 급감 (자사주 마법 차단)
- KOSPI 시총 자사주 비중 7% → 3-4% 감소 예상

### 장기 (2027.09 이후)
- 자사주 비중 글로벌 표준 (1-2%) 정착
- 자사주 마법 사실상 소멸
- 주주환원 직접성 강화 (배당 + 의무소각)
