# 최대주주 변경 공시

## 개요

최대주주가 바뀌거나 바뀌게 되는 계약(주식양수도, 담보제공, 합병 등)이 체결되면 제출하는 의무 공시. 경영권 이전의 핵심 신호.

여러 형태로 분리 제출됨:
- 최대주주변경을 수반하는 주식양수도 계약 체결
- 최대주주변경을 수반하는 주식담보제공 계약 체결
- 최대주주변경 (단순 보고)
- 최대주주변경을 수반하는 주식양수도 계약 종료/해제

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=I (거래소공시) 또는 B (주요사항보고)
- **DART API**: 전용 구조화 API 없음. `list.json` + 제목 키워드 매칭
- **report_nm 패턴**:
  - `최대주주변경을수반하는주식양수도계약체결`
  - `[기재정정]최대주주변경을수반하는주식양수도계약체결`
  - `최대주주변경을수반하는주식담보제공계약체결`
  - `최대주주변경` (단순 변경 보고)
- **의무 여부**: 의무
- **공시 주기**: 수시 (계약 체결/변경 발생 즉시)

## 전체 문서 구조 (트리)

표준 서식 (주식양수도 계약 기준):

```
[회사명]/최대주주변경을수반하는주식양수도계약체결/...
│
├── 1. 양도인 (기존 최대주주)
│   ├── 성명/회사명
│   ├── 발행회사와의 관계
│   ├── 양도 전 보유 주식수 + 비율
│   └── 양도 후 잔여 주식수 + 비율
│
├── 2. 양수인 (새 최대주주)
│   ├── 성명/회사명
│   ├── 발행회사와의 관계
│   ├── 양수 전 보유 주식수
│   └── 양수 후 보유 주식수 + 비율
│
├── 3. 양수도 거래
│   ├── 양수도 주식수
│   ├── 주당 가격
│   ├── 총 양수도 금액
│   ├── 자금조달 방법
│   ├── 양수도 일정 (계약일, 잔금일, 양수도일)
│   └── 양수도 조건
│
├── 4. 변경 후 최대주주
│   ├── 보유 주식수 + 지분율
│   └── 특수관계인 합산 지분율
│
└── 5. 거래 목적                                  (자유기재)
```

## API 필드 vs 원문 대응

별도 구조화 API 없음. `list.json` 응답 기본 필드만:

| API 필드 | 의미 | 비고 |
|----------|------|------|
| `corp_code`, `corp_name` | 발행회사 | |
| `report_nm` | 공시 제목 | 키워드 매칭 (양수도/담보/단순변경 구분) |
| `rcept_no` | 접수번호 | |
| `rcept_dt` | 접수일 | |
| `flr_nm` | 제출인 | 보통 발행회사명, 양수인이 별도 보고 시 양수인명 |

**원문 상세**: 양도인/양수인/거래조건/지분율 변동은 모두 원문(get_document) 파싱 필요.

## OPM에서의 활용

- **`screen_events(event_type="major_shareholder_change")`**: 최근 변경 발생 기업 시장 전체 조회
- **`ownership_structure(scope="changes")`**: 동일 기업의 [[최대주주등소유주식변동신고서]]와 시계열 통합
- **`evidence(rcept_no=...)`**: 원문 viewer URL

## 거버넌스 분석 포인트

1. **양도 형태 판단** (제목 키워드):
   - **주식양수도 계약**: 실제 매각, 경영권 이전 진행
   - **주식담보제공 계약**: 차입 담보, 미래 경영권 이전 위험
   - **단순 최대주주변경**: 합병/분할 등으로 자동 변경
2. **양수인 정체**:
   - 사모펀드 → 단기 차익 추구 (경영권 재매각 가능성)
   - 동종업계 → 산업 재편
   - 외국인 / 외국 자본 → 규제/정치 이슈 가능
3. **지분 변동 폭**:
   - 50% 이상 양수도 → 완전 경영권 이전
   - 30~50% → 지배력 확보 단계
   - 20~30% → 지배력 도전 가능성, 분쟁 신호
4. **계약 종료/해제 공시**: 협상 결렬 시 [기재정정] 또는 별도 종료 공시. 경영권 분쟁 신호.
5. **담보제공 → 양수도 전환**: 차입 담보가 만기에 처분되면 강제 경영권 이전. 채권자 최대주주화 위험.
