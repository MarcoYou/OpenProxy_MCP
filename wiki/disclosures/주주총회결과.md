## 개요

주주총회 종료 후 투표 결과를 공시하는 의무 공시. 안건별 찬반율, [[참석률]] 역산의 기반.

## 소스

- **제출처**: [[KRX-KIND]] (거래소 공시)
- **의무 여부**: 의무
- **공시 주기**: 수시 (주총 종료 후 즉시)

## 주요 내용

- 안건별 발행기준/행사기준 찬성률
- [[감사위원-의결권-제한]] 적용 안건 별도 섹션
- [[집중투표]] 결과 (득표율 + 순위)
- 가결/부결 여부

## 전체 문서 구조 (트리)

```
주주총회 결과
│
├── 개최 정보
│   ├── 주총 일시 및 장소
│   ├── 발행주식 총수
│   ├── 참석 주식수 및 참석률  ← 참석률 역산의 기준
│   └── 의장 (대표이사 또는 이사회 지명인)
│
├── 안건별 결과
│   ├── 안건번호 및 안건명
│   ├── 발행기준 찬성률 (= 찬성주식수 / 발행주식 총수)
│   ├── 행사기준 찬성률 (= 찬성주식수 / 참석주식수)
│   ├── 찬성 / 반대 / 기권 주식수
│   └── 가결 / 부결 여부
│
├── 감사위원 선임 안건 (해당 시)
│   └── 감사위원-의결권-제한 적용 별도 행  ← 3% 룰 적용
│
├── 집중투표 결과 (해당 시)
│   └── 후보자별 득표수 및 득표율, 선임 순위
│
└── 특이사항
    └── 수정안 상정, 의결권 분쟁, 주주 발언 등
```

소스: [[KRX-KIND]] (투표결과 HTML 페이지), DART 주주총회결과 공시 — 두 경로 병존
KIND URL: `acptNo` 마지막 두 자리 `"80"` → `"00"` 변환 필요 (KIND 접수번호 규칙)

## API 필드 vs 원문 대응

| 항목 | KIND HTML 위치 | 실제 의미 | 주의사항 |
|------|---------------|-----------|----------|
| 안건번호 | 테이블 1열 | 의안 순번 | 소집공고와 번호 일치 확인 필요 |
| 찬성주식수 | 테이블 내 찬성 열 | 절대 주식수 | 발행기준·행사기준 찬성률 계산의 분자 |
| 발행주식 총수 | 테이블 헤더 or 개최정보 | 기준 총수 | 발행기준 찬성률 계산의 분모 |
| 참석주식수 | 개최정보 섹션 | 실제 행사 주식수 | 행사기준 찬성률 계산의 분모 |
| 가결여부 | 결과 열 | "가결" / "부결" | |
| acptNo | KIND URL 파라미터 | KIND 접수번호 | DART rcept_no 마지막 2자리 교체 필요 |

**참석률 역산 공식**: `참석률 = 발행기준 찬성률 / 행사기준 찬성률`
→ 두 찬성률 모두 KIND HTML에서 파싱. [[참석률]] 개념 페이지 참조.

## OPM에서의 활용

agm_result tool이 [[KRX-KIND]]에서 HTML을 크롤링하여 파싱. [[KIND-주총결과]] 분석에서 상세 설명.

- [[참석률]] 역산: 발행기준 찬성률 / 행사기준 찬성률
- KOSPI 200 전수 크롤링: 199/199 완료
- [[v4-스키마]]의 voteResults 섹션에 통합
