# 회사분할결정

## 개요

이사회에서 회사분할을 결정하면 제출하는 의무 공시. 단순·물적분할(자회사 신설), 인적분할(주주에게 신주 배정) 구분과 함께 분할대상 사업/존속·신설회사 정보/재상장 여부 등이 담긴다. 지주회사 전환 또는 사업부 분리의 핵심 도구.

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=B (주요사항보고)
- **DART API**: `cmpDvDecsn.json` (DS005 주요사항보고서)
- **report_nm 패턴**: `주요사항보고서(회사분할결정)`, `회사분할결정`, `[기재정정]주요사항보고서(회사분할결정)`
- **의무 여부**: 의무
- **공시 주기**: 수시 (이사회 결의 익영업일까지)

## 전체 문서 구조 (트리)

```
주요사항보고서(회사분할결정)
│
├── 1. 분할 형태 및 방법

├── 4. 존속회사 (분할 후 남는 회사)
```

## API 필드 vs 원문 대응

| API 필드 | 원문 위치 | 의미 | 주의사항 |
|----------|-----------|------|----------|
| `bddd` | 9. 이사회 | 이사회 결의일 | |
| `ex_sm_r` | 1. 형태 | 단순·물적분할 / 인적분할 | **물적/인적이 거버넌스 관점 핵심 분기** |
| `dv_mth` | 1. 방법 | 상법 조항 인용 + 분할 절차 | 자유기재 텍스트 |
| `dv_impef` | 1. 영향 | 지배구조 영향 | 단순물적은 "지분율 변동 없음" 표준 문구 |
| `dv_rt` | 2. 비율 | 분할비율 | 단순물적은 "신주 전부 존속회사 배정" 텍스트 |
| `dv_trfbsnprt_cn` | 3. 분할대상 | 사업부문 설명 | 알짜 사업부 분리 vs 부실 사업부 분리 판단 |
| `atdv_excmp_cmpnm` | 4. 존속회사 | 존속회사명 | 보통 분할 전 회사 그대로 |
| `atdv_excmp_atdv_lstmn_atn` | 4. 재상장 | 존속회사 상장유지 | "예"가 일반적 |
| `atdvfdtl_*` | 4. 재무 | 분할 후 존속회사 재무 | |
| `dvfcmp_cmpnm` | 5. 신설회사 | 신설회사명 | "(가칭)" 포함 빈번 |
| `dvfcmp_nbsn_rsl` | 5. 예상매출 | 신설회사 예상 매출 | 분할대상 사업의 직전 매출 |
| `dvfcmp_rlst_atn` | 5. 재상장 | **신설회사 재상장 여부** | **물적분할+신설회사 재상장 = 모회사 주주 가치 희석 이슈** |
| `gmtsck_prd` | 7. 주총 | 주총 예정일 | 단순물적은 주총 생략 가능 |
| `dvdt` | 7. 분할기일 | 분할 효력 발생일 | |
| `abcr_*` | 6. 자본조정 | 인적분할 시 감자비율, 신주배정 일정 | 단순물적은 "-" |

## OPM에서의 활용

- **`corporate_restructuring(scope="split")`**: 분할결정과 분할합병결정을 합쳐 카드 표시. 분할형태/존속·신설회사/재상장 여부를 한눈에.
- **`corporate_restructuring(scope="summary")`**: 4종 통합 timeline에서 신설회사명 + 분할비율 노출.
- **`screen_events`** (향후): `split_decision` event_type 추가 시 시장 전체에서 검색.

## 거버넌스 분석 포인트

1. **물적분할 후 신설회사 재상장**: `ex_sm_r=단순·물적` + `dvfcmp_rlst_atn=예` 패턴 — LG화학 → LG에너지솔루션 사례. 모회사 주주 가치 희석 우려.
2. **인적분할**: 주주에게 신설회사 신주 배정. 지주회사 전환 시 자주 사용.
3. **분할대상 사업의 성격**: 캐시카우 분리(가치 빼내기) vs 부실 사업 분리(정리 목적) 판단.
4. **존속회사 재상장 유지 + 신설회사 비상장**: 지배주주가 신설회사를 100% 비상장으로 보유 → 일감몰아주기 가능 구조.
5. **주총 생략 (단순물적)**: 소수주주 의결권 행사 기회 없음. 주식매수청구권만 보호 장치.
