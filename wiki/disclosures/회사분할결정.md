# 회사분할결정

## 메타

| 항목 | 값 |
|------|---|
| DART API | `cmpDvDecsn.json` (DS005 주요사항보고서) |
| pblntf_ty | B |
| report_nm 패턴 | `주요사항보고서(회사분할결정)` |
| OPM tool | `corporate_restructuring(scope="split")` |

## 핵심 필드

| 필드 | 의미 |
|------|------|
| `bddd` | 이사회 결의일 |
| `ex_sm_r` | 분할형태 (단순·물적분할 / 인적분할 등) |
| `dv_mth` | 분할방법 |
| `dv_impef` | 분할이 지배구조에 미치는 영향 |
| `dv_rt` | 분할비율 (단순물적이면 100% 존속회사) |
| `dv_trfbsnprt_cn` | 분할대상 사업내용 |
| `atdv_excmp_cmpnm` | 존속회사명 |
| `atdv_excmp_atdv_lstmn_atn` | 존속회사 상장 유지 여부 |
| `atdvfdtl_*` | 존속회사 재무 |
| `dvfcmp_cmpnm` | 신설회사명 |
| `dvfcmp_mbsn` | 신설회사 사업내용 |
| `dvfcmp_nbsn_rsl` | 신설회사 예상매출 |
| `dvfcmp_rlst_atn` | 신설회사 재상장 여부 |
| `gmtsck_prd` | 주주총회 예정일 |
| `dvdt` | 분할기일 |

## 거버넌스 분석 포인트

- 인적분할 vs 물적분할: 인적은 주주에게 신설회사 신주 배정, 물적은 존속회사가 100% 보유
- 물적분할 후 신설회사 재상장: 기존 주주 가치 희석 우려 (LG화학 → LG에너지솔루션 사례)
- 분할대상 사업: 캐시카우 분리 vs 부실 사업부 분리 판별
- 존속회사 재상장 유지 + 신설회사 비상장: 지배주주 우호 구조
