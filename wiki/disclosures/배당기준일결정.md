# 배당기준일결정

배당 권리주주를 확정하는 기준일을 이사회에서 결정할 때 제출하는 거래소 수시공시. 2024년 자본시장법 개정으로 **선(先)배당기준일 - 후(後)결의** 방식이 가능해지면서 별도 공시로 분리되는 사례가 급증.

## 개요

| 항목 | 값 |
|------|-----|
| 공시 의무 회사 | 상장사 전체 (배당 실시 회사) |
| 제출처 | KRX-KIND, DART는 `pblntf_ty="I"` |
| 시기 | 이사회결의일 다음 영업일까지 |
| 의무 여부 | 의무 |
| 공시 주기 | 수시 (배당기준일 결정 시) |
| 2024 개정 | 자본시장법 시행령 개정으로 결산일 기준일과 분리 가능 — 배당금 확정 후 기준일 설정 가능 |

## 2024 개정 배경 ([[시간순서-규칙]])

기존: 결산일(12/31) = 배당기준일 → 주주 확정 후 다음해 3월 주총에서 배당금 결정 → 주주는 배당 금액을 모르고 권리 확보 (caveat emptor)

개정 후: 정관 변경 → "이사회가 별도로 정한 날" 가능 → 배당금 결정 후 배당기준일 별도 공고 → **선배당-후결의** → 주주가 배당금 확인 후 매수 가능 ([[시간순서-규칙]])

## 전체 문서 구조 (트리)

```
[회사명]/현금ㆍ현물배당을 위한 주주명부폐쇄(기준일) 결정/([날짜])...

현금ㆍ현물배당을 위한 주주명부폐쇄(기준일) 결정
│
├── 1. 배당구분          결산 | 중간(분기)
├── 2. 주주명부폐쇄(기준일)
│   ├── 시작일            -  (폐쇄 없이 기준일만 쓰는 경우 -)
│   ├── 종료일            -
│   └── 기준일            YYYY-MM-DD
├── 3. 주주명부폐쇄(기준일) 목적    권리주주 확정
├── 4. 이사회결의일(결정일) YYYY-MM-DD
│   ├── 사외이사 참석여부    참석(명) N / 불참(명) N
│   └── 감사(사외이사가 아닌 감사위원) 참석여부  -
├── 5. 기타 투자판단과 관련한 중요사항
└── ※ 관련공시  (배당기준일 안내 선행공시 참조)
```

## 별도 공시: 주주명부폐쇄기간또는기준일설정 (배당 무관)

배당과 무관하게 **정기주총 의결권 주주 확정** 또는 **분배금 지급**(투자회사 — 펀드/인프라)을 위한 기준일 설정. 동일 서식 사용 — `설정사유` 텍스트로 구분 필수.

```
주주명부폐쇄기간 또는 기준일 설정
│
├── 1. 기준일            YYYY-MM-DD
├── 2. 명의개서정지기간
│   ├── 시작일            - (대부분 폐쇄 없음)
│   └── 종료일            -
├── 3. 설정사유          (자유기재)
│       예) "제89기 정기주주총회 권리주주 확정"
│       예) "2025년 하반기 분배금 지급을 위한 권리주주 확정"
├── 4. 기타 투자판단에 참고할 사항
└── ※ 관련 공시
```

**주의**: 12월에 기준일 사전 공고 → 1-2월에 배당금액 결정 (현금배당결정 공시) 순서 빈번.

## DART API 매핑

- **API**: 별도 구조화 API 없음
- **report_nm 패턴**:
  - `현금ㆍ현물배당을위한주주명부폐쇄(기준일)결정`
  - `주주명부폐쇄기간또는기준일설정` (주총용 + 분배금용 혼재)
  - `[기재정정]현금ㆍ현물배당을위한주주명부폐쇄(기준일)결정`

## OPM 활용

- **`div_search(ticker)`** — 배당기준일 결정 포함 검색
- **분기배당 기업 추적**: POSCO홀딩스, 현대차, KB금융, 삼성전자 등 — 분기마다 기준일+배당결정 2종 동시 제출
- **선배당-후결의 추적**: 동일 회사 동일 회차에서 (1) 배당기준일결정 → (2) 현금배당결정 순서로 별도 공시 시 신정관 적용 회사로 분류
- **주총용 기준일 vs 배당 기준일 구분**: 동일 서식 — `설정사유` 텍스트 파싱 필수

## 거버넌스 분석 포인트

- **선배당-후결의 채택 여부**: 정관 변경 후 채택한 회사는 주주친화적 시그널 (밸류업 지표 일부)
- **분기배당 정착 여부**: 분기마다 기준일결정 4건 + 배당결정 4건 = 정착 회사
- **자회사판 (주요경영사항)**: 금융지주가 비상장 자회사 배당기준일 대신 공시 — KOSPI 200 모회사 입장에서 시장 이벤트 아님
