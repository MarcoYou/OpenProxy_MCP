# 분기배당결정

분기별 중간배당을 결정한 후 제출하는 거래소 수시공시. 형식은 [[현금배당결정]]과 동일하나 **`1. 배당구분 = 분기배당` 또는 `중간(분기)`** 로 구분되며, 정기 결산배당과 별도로 추적 필요. 분기배당 기업은 연 4회 (Q1/반기/Q3/결산) 별도 공시 제출.

## 개요

| 항목 | 값 |
|------|-----|
| 공시 의무 회사 | 분기배당 정관 채택 회사 (정관 §제N조 분기배당) |
| 제출처 | KRX-KIND, DART는 `pblntf_ty="I"` |
| 시기 | 분기 말일 후 이사회결의 → 다음 영업일까지 |
| 의무 여부 | 의무 (분기배당 결정 시) |
| 공시 주기 | 수시 (분기마다 — 연 최대 4회) |
| 한 회사 / 연 | 정착 회사: Q1/반기/Q3/결산 = 4건 |

## 핵심 데이터 항목

[[현금배당결정]]과 **동일한 서식** 사용. 차이점만 명시:

| 필드 (한글) | 분기배당 특이사항 |
|---|---|
| 1. 배당구분 | `분기배당` 또는 `중간(분기)` (서식에 따라 표기 다름) |
| 2. 배당종류 | `현금배당` (분기배당은 현금만, 현물 사례 거의 없음) |
| 3. 1주당 배당금 | **해당 분기분만** (누적 X) — 연간 합산은 사업보고서 alotMatter 별도 |
| 6. 배당기준일 | 분기 말일 (3/31, 6/30, 9/30, 12/31) — 정관에 명시 |
| 7. 배당금지급 예정일 | 통상 분기 말 후 1-2개월 |
| 8. 주주총회 개최여부 | **미개최** (분기배당은 주총 승인 불요, 이사회 결의로 확정) |
| 10. 이사회결의일 | 분기 말일 직후 (예: Q1 → 4월) |

## 정기배당 vs 분기배당 비교

| 항목 | 결산배당 | 분기배당 |
|---|---|---|
| 결정 주체 | 주주총회 (이사회 결의 → 주총 승인) | 이사회 (단독 결정) |
| 주총 개최여부 | 개최 | 미개최 |
| 기준일 | 결산일 (12/31) 또는 별도 | 분기 말일 |
| 횟수 | 연 1회 | 연 1-3회 (Q1/반기/Q3) |
| 정관 근거 | 상법 §462 | 상법 §462의3 + 정관 명시 |
| 한도 | 이익잉여금 한도 | 직전 결산 후 잉여금 한도 (보수적) |

## 연간 DPS 산정

```
연간 DPS = 1Q DPS + 반기 DPS + 3Q DPS + 결산 DPS
```

- **각 분기 배당은 해당 분기분만** (누적이 아님)
- DART reprt_code로 구분: 11013(1Q), 11012(반기), 11014(3Q), 11011(결산)
- 사업보고서 alotMatter는 **연간 합산** — 회차별 공시와 합치 검증 필요

## 예시: 삼성전자 2025

| 회차 | 배당구분 | DPS | 배당기준일 | 이사회결의일 |
|------|---------|-----|-----------|------------|
| Q1 | 분기배당 | 366원 | 2025-03-31 | 2025-04-30 |
| 반기 | 분기배당 | 366원 | 2025-06-30 | 2025-07-31 |
| Q3 | 분기배당 | 370원 | 2025-09-30 | 2025-10-31 |
| 결산 | 결산배당 | 566원 | 2025-12-31 | 2026-01-29 |
| **합계** | - | **1,668원** | - | - |

→ alotMatter `주당 현금배당금(원) = 1,668`

## DART API 매핑

- **API**: 별도 구조화 API 없음 (`list.json` + 키워드 검색)
- **report_nm 패턴**: `현금ㆍ현물배당결정` (정기와 동일 — `1. 배당구분` 본문 파싱으로 분기 판별)
- **제출 시기**: 매 분기 말일 후 1개월 이내 이사회결의 → 다음날 공시
- **자회사판**: 금융지주(KB/하나/신한/우리/메리츠) 비상장 증권 자회사 분기배당 → `(자회사의 주요경영사항)` 버전 별도 제출

## OPM 활용

- **`div_search(ticker)`** — `1. 배당구분 = 분기배당` 필터링
- **`div_history(ticker, years)`** — 연간 DPS = 1Q + 반기 + 3Q + 결산 합산
- **flow vs stock 표시**: 개별 분기 DPS (flow) + 연간 합산 (stock) 모두 노출
- **분기배당 기업 식별**: 동일 회사 같은 사업연도 내 `현금ㆍ현물배당결정` 공시가 2건 이상 → 분기배당 기업
- **분기배당 도입 시점**: 정관변경 안건 (분기배당 조항 신설) 추적 → [[정관변경]]

## 거버넌스 분석 포인트

- **분기배당 정착 여부**: KOSPI 200 중 분기배당 정착 회사 ~30개 (2025 기준)
  - 대표: 삼성전자, POSCO홀딩스, 현대차, 기아, KB금융, 하나금융, 신한금융, 우리금융, 메리츠금융지주, SK텔레콤, KT, KT&G
- **분기 DPS 안정성**: 매 분기 일정 → 안정적 캐시플로우 신호
- **분기→결산 비중**: 결산 비중이 압도적으로 높으면 형식적 분기배당 (보여주기식)
- **반기/Q3 누락**: 분기배당 도입했으나 일부 분기 누락 → 주주환원 의지 약함
