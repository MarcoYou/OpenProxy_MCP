# 회사합병결정

## 메타

| 항목 | 값 |
|------|---|
| DART API | `cmpMgDecsn.json` (DS005 주요사항보고서) |
| pblntf_ty | B |
| report_nm 패턴 | `주요사항보고서(회사합병결정)` |
| OPM tool | `corporate_restructuring(scope="merger")` |

## 핵심 필드

| 필드 | 의미 |
|------|------|
| `bddd` | 이사회 결의일 |
| `mg_stn` | 합병형태 (소규모합병 / 일반합병) |
| `mg_mth` | 합병방법 (흡수합병 / 신설합병) |
| `mg_pp` | 합병목적 |
| `mg_rt` | 합병비율 |
| `mg_rt_bs` | 비율 산정근거 |
| `mgnstk_ostk_cnt` | 합병신주(보통주) 수 |
| `mgptncmp_cmpnm` | 합병 상대회사명 |
| `mgptncmp_mbsn` | 상대회사 사업내용 |
| `mgptncmp_rl_cmpn` | 상대회사 관계 (자회사 등) |
| `rbsnfdtl_*` | 상대회사 재무 (자산/부채/자본/매출/순이익) |
| `exevl_intn` | 외부평가 기관 |
| `exevl_op` | 외부평가 의견 |
| `eadtat_op` | 상대회사 감사의견 |
| `aprskh_plnprc` | 주식매수청구권 가격 |
| `popt_ctr_atn` | 풋옵션 약정 여부 |

## 거버넌스 분석 포인트

- 합병비율: 소수주주 권리 보호 → 비율 산정의 합리성
- 매수청구권: 반대주주 보호 장치
- 외부평가 의견: 가격 적정성
- 풋옵션: 일방적 유리한 조건 여부
