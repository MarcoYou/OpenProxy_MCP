# 주식교환·이전 결정

## 메타

| 항목 | 값 |
|------|---|
| DART API | `stkExtrDecsn.json` (DS005 주요사항보고서) |
| pblntf_ty | B |
| report_nm 패턴 | `주요사항보고서(주식교환ㆍ이전결정)` |
| OPM tool | `corporate_restructuring(scope="share_exchange")` |

## 개념

- **주식교환**: 두 회사 주주가 서로 주식을 맞바꾸어 한 회사가 다른 회사의 100% 모회사가 됨
- **주식이전**: 주주가 주식을 새로 설립되는 모회사에 이전하고 모회사 신주를 받음

→ 지주회사 전환의 정형화된 도구

## 거버넌스 분석 포인트

- 자회사 100% 편입: 비상장 전환 → 소수주주 권리 소멸 + 매수청구권 가격
- 교환비율: 자회사 가치 평가의 합리성 (이마트 → 신세계건설 100% 자회사 사례)
- 외부평가 의견 vs 자체 평가: 회사간 가치 협상력
