# 경영권분쟁소송

## 개요

[[소송등의제기]] 중 **경영권 분쟁과 직접 관련**된 소송을 명시적으로 분류한 공시. 회사·이사·주주 사이의 지배력 다툼이 법적 분쟁으로 비화한 시점.

별도 분리 제목군으로, OPM에서는 일반 [[소송등의제기]]와 구분해 더 강한 신호로 처리.

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=I (거래소공시) 또는 B (주요사항보고)
- **DART API**: 전용 구조화 API 없음. `list.json` + 제목 키워드 매칭
- **report_nm 패턴**: `소송등의제기ㆍ신청(경영권분쟁소송)`, `경영권분쟁소송`
- **의무 여부**: 의무
- **공시 주기**: 수시 (사건 발생 즉시)

## 키워드 (OPM 사용)

```python
keywords = ("경영권분쟁소송",)
```

이 키워드가 제목에 명시되면 회사가 자체 분류로 "경영권 분쟁"이라고 인정한 사건. 더 강한 시그널.

## 전체 문서 구조 (트리)

[[소송등의제기]]와 동일 양식. 차이는 제목에 "경영권분쟁소송" 명시.

```
[회사명]/소송등의제기ㆍ신청(경영권분쟁소송)/...
│
├── 1. 사건명                         (주총 결의 무효, 직무집행정지 등)
├── 2. 원고                           ★ 분쟁 상대방 식별
├── 3. 피고                           ★ 회사 또는 경영진
├── 4. 청구취지                        (요구 사항)
├── 5. 청구원인                        (분쟁 사유)
├── 6. 관할 법원
├── 7. 사건번호
├── 8. 사건 발생일자
├── 9. 소송가액
├── 10. 회사에 미치는 영향
└── 11. 향후 대응 방안
```

## API 필드 vs 원문 대응

[[소송등의제기]]와 동일.

| API 필드 | 의미 | 비고 |
|----------|------|------|
| `corp_code`, `corp_name` | 회사 | |
| `report_nm` | 공시 제목 | "경영권분쟁소송" 키워드 포함 |
| `rcept_no`, `rcept_dt` | 접수번호/접수일 | |

원문(get_document) 파싱으로 원고/피고/청구취지 등 상세 추출.

## OPM에서의 활용

- **`proxy_contest(scope="litigation")`**: 일반 소송과 합쳐 timeline 생성. `event_type="management_dispute"` 분류로 표시.
- **`proxy_contest(scope="summary")`**: 분쟁 신호 카운트에 강한 가중치
- **`screen_events(event_type="management_dispute")`**: 시장 전체 경영권 분쟁 발생 기업 검색
- **`build_campaign_brief`**: 캠페인 사실 브리프 timeline에 핵심 사건으로 포함

## 거버넌스 분석 포인트

### 사건 유형별 분쟁 단계

| 사건 유형 | 분쟁 단계 |
|----------|----------|
| **회계장부 열람·등사** | 초기 — 정보 접근권 확보 |
| **주주명부 열람** | 초기·중기 — 위임장 권유 사전 단계 |
| **임시주주총회 소집허가** | 중기 — 소수주주가 주총 진행권 확보 |
| **주식발행금지·신주배정금지** | 중기 — 우호 지분 형성 차단 |
| **이사 직무집행정지** | 중·후기 — 분쟁 가속, 경영 마비 |
| **주주총회 결의 무효·취소** | 후기 — 주총 결과 뒤집기 |

### 원고 정체 분석

- **소수주주 / 행동주의 펀드**: 가장 흔한 패턴. ESG 펀드, 헤지펀드 등.
- **전임 경영진**: 내부 분쟁 (삼성가, 한진가 사례 등)
- **공동 보유자**: [[5%-대량보유]] 공동보유 약정 깨짐
- **모회사 주주**: 자회사 IPO 분쟁 등

### 후속 신호 모니터링

경영권분쟁소송 공시 발생 후 자주 따라오는 후속 공시:
- [[위임장권유참고서류]] 제출 (양측)
- [[대량보유상황보고서]] (보유목적: 경영참여)
- [[주주총회소집공고]] 임시주총
- [[자기주식결정]] (방어 목적 취득/처분)
