## 개요

이사회에서 현금배당을 결정한 후 공시하는 거래소 공시. DPS, 배당기준일, 지급일, [[특별배당]] 여부 등 배당 이벤트의 핵심 정보.

## 소스

- **제출처**: [[KRX-KIND]] (거래소 공시, pblntf_ty=I)
- **의무 여부**: 의무 (배당 결정 시)
- **공시 주기**: 수시 (배당 결정 시)

## 주요 내용

- 1주당 배당금 (DPS)
- 배당기준일
- 배당금 지급일
- [[시가배당률]]
- [[특별배당]] 여부
- 배당 대상 주식 종류 (보통주/우선주)

## 전체 문서 구조 (트리)

```
현금(현물)배당결정
│
├── 배당 개요
│   ├── 배당 종류 (현금 / 현물 / 주식배당)
│   ├── 1주당 배당금 — 보통주  ← DPS 핵심 필드
│   ├── 1주당 배당금 — 우선주 (해당 시)
│   ├── 시가배당률 (보통주 / 우선주)  ← 시가배당률 필드
│   └── 배당금 총액
│
├── 배당 일정
│   ├── 배당 기준일 (주주 확정일)
│   ├── 배당금 지급 예정일
│   └── 주주총회 예정일 (결산배당의 경우)
│
└── 배당 근거
    ├── 이사회 결의일
    └── 배당 구분 (결산배당 / 분기배당 / 중간배당 / 특별배당)
```

소스: KRX 거래소 표준 서식 (KIND), DART `pblntf_ty="I"` 검색으로도 조회 가능

## API 필드 vs 원문 대응

| KIND / DART 필드 | 원문 위치 | 실제 의미 | 주의사항 |
|-----------------|-----------|-----------|----------|
| `divPfmAmt` (유사) | 배당 개요 > 1주당 배당금 (보통주) | 보통주 DPS (원) | 회차별 DPS — 사업보고서 연간 합산과 다름 |
| `prefDivPfmAmt` (유사) | 배당 개요 > 1주당 배당금 (우선주) | 우선주 DPS (원) | |
| `mkPrcDivrt` (유사) | 배당 개요 > 시가배당률 | 시가배당률 (%) | 공시 당시 주가 기준 |
| `divTotAmt` (유사) | 배당 개요 > 배당금 총액 | 총 배당금 지급액 | |
| `divRcdDt` (유사) | 배당 일정 > 배당 기준일 | 주주 확정 기준일 | 분기배당은 분기 말 |
| `divPayDt` (유사) | 배당 일정 > 지급 예정일 | 실제 지급일 | |
| `pblntf_ty` | - | 공시 유형 | `"I"` = 현금배당결정 (DART 기준) |

**KIND HTML 직접 파싱 시**: 표준 서식 필드명과 실제 테이블 셀 순서가 다를 수 있음.
→ 보통주 DPS, 우선주 DPS, 시가배당률 순서 확인 후 파싱할 것.

## OPM에서의 활용

- div_detail: 현금배당결정 공시에서 해당 회차분 DPS만 추출

- [[분기배당]] 기업은 분기마다 별도 현금배당결정 공시 발생
- [[div-tool-rule]]에서 DPS 데이터 소스 우선순위 정의
