# 주식배당결정

이사회에서 주식(신주)으로 배당을 지급하기로 결정한 후 제출하는 거래소 수시공시. 현금배당과 병행 또는 단독 지급. 주총 안건으로 상정 — 주총 결과에 따라 변경 가능.

## 개요

| 항목 | 값 |
|------|-----|
| 공시 의무 회사 | 상장사 전체 |
| 제출처 | KRX-KIND, DART는 `pblntf_ty="I"` |
| 시기 | 이사회결의일 다음 영업일까지 |
| 의무 여부 | 의무 (자본시장법 §161 ① 7호) |
| 공시 주기 | 수시 (주식배당 결정 시) |
| 비고 | 상법 §462의2: 이익배당총액의 1/2 한도 + 정관에 근거 필요 |

## DART API 매핑

- **API**: 별도 구조화 API 없음 (`list.json` + 키워드 검색)
- **report_nm 패턴**: `주식배당결정`, `[기재정정]주식배당결정`
- **본문 파싱**: `evidence(rcept_no)` → DART 뷰어에서 추출

## OPM 활용

- **`div_search(ticker)`** — 주식배당결정 포함 검색
- **희석 EPS 보정**: 주식배당 신주 수 반영 — 발행주식총수 변동 시점 확인
- **연간 DPS 합산 미포함**: 주식배당은 별도 행으로 표시 (현금 DPS와 합산 X)
- **자기주식 제외 산정**: `발행주식총수`는 자기주식 포함 총수 — 실제 배당 대상은 공제 후

## 거버넌스 분석 포인트

- **현금 vs 주식배당**: 현금 부족 + 주가 부양 의도일 가능성
- **희석 효과**: 발행주식수 증가 → EPS 하락 (주가 변동 미미하면 P/E 변동)
- **단주 처리**: 1주 미만 단수주는 통상 현금 환산
- **정관 근거 누락**: 정관에 주식배당 조항 없으면 임시 정관변경 안건 동반
