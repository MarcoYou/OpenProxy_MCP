# 자기주식취득결정

회사가 직접 시장에서 자기주식을 매입하기로 결정하는 공시. 주주환원(소각 전제) 또는 경영권 방어 수단. **2026.03 신법 시행으로 취득 후 1년 내 의무소각** — 취득 단계에서 소각 commitment 명시 여부가 핵심 분석 포인트.

## 개요

| 항목 | 값 |
|------|-----|
| 공시 의무 회사 | 상장사 전체 (자기주식 취득 결정 시) |
| 제출처 | DART (주요사항보고서 — DS005 그룹) |
| 시기 | 이사회결의일 다음 영업일까지 |
| 의무 여부 | 의무 (상법 §341 + 자본시장법 §161) |
| 공시 주기 | 수시 |
| API | `tsstkAqDecsn.json` (DART OpenAPI 구조화 제공) |
| pblntf_ty | B (주요사항보고서) |
| 한도 | 배당가능이익 한도 (상법 §341 ①) |

## DART API 매핑

- **endpoint**: `https://opendart.fss.or.kr/api/tsstkAqDecsn.json`
- **그룹**: DS005 (자기주식 결정)
- **파라미터**: `corp_code`, `bgn_de`, `end_de`
- **응답 구조**: `list[]` 배열 — 위 필드 모두 포함
- **report_nm 패턴**: `주요사항보고서(자기주식취득결정)`, `[기재정정]` 동반 가능

## OPM 활용

- **`treasury_share(scope="acquisition")`** — 취득결정만 조회 (신탁체결 포함)
- **`treasury_share(scope="events")`** — 5종 통합 timeline
- **`treasury_share(scope="summary")`** — 집계 + 최신 5건
- **`for_cancelation` 자동 검출** (services/treasury_share.py): `aq_pp`에 "소각" 키워드 포함 시 `for_cancelation=True` 플래그
- **`screen_events(event_type="treasury_acquire")`** — 시장 전체 검색

## 거버넌스 분석 포인트

### 취득 목적별 분류 (`aq_pp`)
| 목적 | 분석 의미 | OPM guideline default |
|------|---------|----------------------|
| 주주가치 제고 및 소각 | 강한 주주환원 신호 | for |
| 주가 안정화 | 주주환원 + 시그널 | for/review |
| 임직원 보상 (스톡옵션 행사 대응) | 보수 정책 일부 | review |
| 경영권 방어 | 우호 지분 확보 (논쟁적) | against (1년 내 소각 commitment 없으면) |
| 합병 대가 | M&A 거래 일부 | review |

### 2026.03 신법 영향 (의무소각)
- **`aq_pp`에 "소각" 명시**: for (commitment 확보)
- **`aq_pp`에 "소각" 미명시**: against — 1년 내 소각 의무 위반 시 처분 안건 against
- **취득기간 만료 + 소각결정 누락**: 소각 의무 위반 알람

### 취득 방법 (`aq_mth`)
- **장내매수**: 시장 영향 직접
- **장외매수**: 특정 매도자와 협의 — 가격 불투명
- **공개매수**: 대규모, 공정 가격
- **신탁** → [[자기주식신탁결정]] 별도 공시
