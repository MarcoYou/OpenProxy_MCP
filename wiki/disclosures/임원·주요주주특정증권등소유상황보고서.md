# 임원·주요주주 특정증권등 소유상황보고서

## 개요

회사의 임원(이사·감사·집행임원) 또는 주요주주(10% 이상 보유자)가 본인 또는 특별관계인의 회사 발행 특정증권(주식, CB, BW, EB 등) 소유 현황과 변동을 보고하는 의무 공시.

[[대량보유상황보고서]](5% 보고)와 별개의 인사이더 보고 의무.

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=D (지분공시)
- **DART API**: `elestock.json` (DS004 지분공시)
- **report_nm 패턴**: `임원ㆍ주요주주특정증권등소유상황보고서`, `[기재정정]임원ㆍ주요주주특정증권등소유상황보고서`
- **의무 여부**: 의무
- **공시 주기**: 수시 (보유 시 5영업일 이내, 변동 시 5영업일 이내)

## 전체 문서 구조 (트리)

```
임원ㆍ주요주주 특정증권등 소유상황보고서
│
├── 1. 보고자 정보
│   ├── 성명                                 ← repror
│   ├── 임원 등기/비등기 구분                  ← isu_exctv_rgist_at
│   ├── 직책                                  ← isu_exctv_ofcps
│   └── 주요주주 여부                          ← isu_main_shrholdr
│
├── 2. 특정증권등 소유 현황
│   ├── 종류 (보통주/우선주/CB/BW/EB)
│   ├── 보유 수량                              ← sp_stock_lmp_cnt
│   ├── 증감 수량                              ← sp_stock_lmp_irds_cnt
│   ├── 보유 비율 (%)                          ← sp_stock_lmp_rate
│   └── 증감 비율 (%)                          ← sp_stock_lmp_irds_rate
│
├── 3. 특별관계인 보유 현황 (해당 시)
│   └── 친족/계열사 등 별도 항목
│
└── 4. 변동 사유
    └── 매수/매도/상속/증여/스톡옵션 행사 등
```

## API 필드 vs 원문 대응

| API 필드 | 원문 위치 | 의미 | 주의사항 |
|----------|-----------|------|----------|
| `rcept_no` | 표지 | 접수번호 | |
| `rcept_dt` | 표지 | 접수일자 | |
| `corp_code`, `corp_name` | 표지 | 발행회사 | |
| `repror` | 1. 보고자 | 보고자 성명 | 본인 보고 + 특별관계인 별도 row |
| `isu_exctv_rgist_at` | 1. 임원구분 | 등기임원 / 비등기임원 | |
| `isu_exctv_ofcps` | 1. 직책 | 사장, 상무, 이사 등 | |
| `isu_main_shrholdr` | 1. 주요주주 | "예" / "-" | 10%+ 보유자 |
| `sp_stock_lmp_cnt` | 2. 보유 | 특정증권 보유 수량 | 단위: 주 |
| `sp_stock_lmp_irds_cnt` | 2. 증감 | 보고 시점 증감 수량 | + 매수, - 매도 |
| `sp_stock_lmp_rate` | 2. 비율 | 보유 비율 (%) | 보통 매우 작음 (임원은 0~1%) |
| `sp_stock_lmp_irds_rate` | 2. 증감비율 | 증감 비율 (%) | |

## OPM에서의 활용

- **`ownership_structure(scope="changes")`**: 사업보고서 임원 보유 현황과 통합
- **`screen_events(event_type="executive_ownership")`**: 최근 임원 보고 발생 기업 검색
- **`evidence(rcept_no=...)`**: 원문 viewer URL

## 거버넌스 분석 포인트

1. **인사이더 거래 신호**: 임원의 매수 = 회사 가치 자신감 / 매도 = 부정적 시그널 (단, 자금 사정·세금·은퇴 등 다양한 사유)
2. **지속적 매도**: 동일 임원의 반복 매도 → 내부 정보 활용 의심 (불공정거래 조사 대상)
3. **스톡옵션 행사 후 즉시 매도**: 행사가-시가 차익 실현. 보유 의지 약함.
4. **친족·계열사 합산 (특별관계인)**: 형식상 임원 본인 보유는 적어도 친족 포함 합계는 큰 케이스 → 실질 지배력 평가
5. **대표이사·CFO 매매**: 일반 임원보다 시그널 강도 높음. `isu_exctv_ofcps`가 "대표이사", "사장" 등 핵심 직책인지 확인
6. **사외이사 보유**: 사외이사가 자사주를 다량 보유하면 독립성 의문
