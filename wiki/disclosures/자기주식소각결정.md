# 자기주식소각결정

보유 자기주식을 영구적으로 소각하여 발행주식수를 감소시키는 결정 공시. **가장 강한 주주환원 신호** (배당과 달리 EPS 자동 상승). 2026.03 신법 시행으로 자기주식 취득 시 1년 내 소각이 원칙 — 소각결정은 더 이상 자율이 아닌 **준의무화**.

## 개요

| 항목 | 값 |
|------|-----|
| 공시 의무 회사 | 상장사 전체 (자기주식 소각 결정 시) |
| 제출처 | DART (거래소공시 — `pblntf_ty="I"`), KIND |
| 시기 | 이사회결의일 다음 영업일까지 |
| 의무 여부 | 의무 (자본시장법 §161 + 거래소 공시규정) |
| 공시 주기 | 수시 |
| API | **별도 구조화 API 없음** — `list.json` + 키워드 검색 |
| pblntf_ty | I (거래소공시) |
| 결의 | **이사회결의** (배당가능이익 한도 내 소각 — 상법 §343 ① 단서) |
| 자본금 감소 시 | 주주총회 특별결의 + 채권자 보호 절차 (감자결정과 동일) |

## DART API 매핑

- **API**: 별도 구조화 API 없음 — `list.json` (`pblntf_ty="I"` + `corp_code` + 기간 + 키워드)
- **검색 키워드** (OPM 사용):
  ```python
  ("주식소각결정", "자기주식소각결정", "자사주소각결정", "자기주식소각")
  ```
- **report_nm 패턴**:
  - `주식소각결정` (가장 일반적)
  - `[기재정정]주식소각결정`
  - 주의: `자기주식소각결정`이 아닌 **`주식소각결정`**으로 등록되는 경우가 다수

## OPM 활용

- **`treasury_share(scope="cancelation")`** — 소각결정만 조회 (본문 파싱 후 소각 주식수·금액·방법 노출)
- **`treasury_share(scope="events")`** — 5종 통합 timeline
- **소각은 회계 정리 단계** — 매입 시점에 이미 현금이 나간 상태이므로 CSR(한국식 주주환원율) 분자에는 사용하지 않는다. CSR 분자는 [[자기주식취득결정]](acquire). 자사주 소각 자체는 발행주식수 감소(EPS 상승) 효과만 정책 신호로 분석.
- 주주환원율 산출은 **`dividend(scope="cash_shareholder_return")`** (한국식 CSR — 배당+자사주 매입) 또는 **`dividend(scope="total_shareholder_return")`** (글로벌 TSR — 주가변동+배당)
- **`screen_events(event_type="treasury_retire")`** — 시장 전체 검색
- **`evidence(rcept_no)`** — 본문 다운로드 (구조화 데이터 부족 시)

### 본문 파싱 (2026-04-29 강화)

별도 구조화 API가 없으므로 list.json 메타 + 본문 정규식 파싱 (`_parse_cancelation_body`):

- 소각 주식수: 단일 종류 ("소각할 주식 수 (주) X") + 보통주+종류주식 분리 표 (삼성전자 케이스) 모두 지원
- 소각 금액: "소각예정금액(원) X" 패턴 (KT&G 1조원+ / 삼성전자 3조원 검증)
- [기재정정] dedupe: (board_date, amount, shares) 튜플 기준 latest만 합산

## 거버넌스 분석 포인트

### 소각 vs 취득 후 보유 구분
- **취득만 하고 보유**: 잠재적 우호 지분 (의결권 없으나 처분 가능성)
- **소각**: 영구 환원 → 발행주식수 감소 → EPS/BPS 자동 상승

### 소각 방법별 분석
| 방법 | 자본 영향 | 결의 | 채권자 보호 |
|------|---------|------|----------|
| 이익잉여금 소각 | 자본금 동일, 이익잉여금 감소 | 이사회 | 불요 |
| 자본금 감소 | 자본금 감소, 이익잉여금 동일 | 주총 특별결의 | 필요 (1개월 공고) |

→ **이익잉여금 소각이 압도적 다수** (간소함)

### 소각 규모 평가
- 발행주식 대비 1% 이상: 의미 있는 환원
- 시가총액 대비 5% 이상: 강한 신호
- KT&G: 2024-2025년 1조원+ 소각 → 적극 주주환원 사례

### 2026.03 신법 영향 (의무소각)
- **신법 핵심**: 취득 → 1년 내 소각 의무
- **취득과 동시 소각 commitment**: [[자기주식취득결정]]에서 `aq_pp`에 "소각" 명시 → for
- **취득기간 만료 + 소각 누락 → 대상 회사 워치리스트**: OPM에서 자동 알람
- **OPM guideline default**:
  - 소각결정 (1년 내 소각 commitment 준수): for
  - 소각 누락 시 처분 안건: against
