## 6. 2026 신법 영향 (자기주식 1년 의무소각)

### 6.1 신법 핵심 ([[자기주식의무소각-2026신법]])

- **시행일**: 2026-03-06
- **근거**: 상법 §341 (취득 사유) + §342 (처분/소각 의무) 개정
- **의무**: 자기주식 취득 후 **1년 내 소각** (원칙) / 합리적 사유 + 외부 검증 후 처분 (예외)
- **기존 보유분**: 시행일 + 6개월 유예 (2026.09.06) → 1년 내 (2027.09.06)까지 처리
- **위반 효과**: OPM guideline에서 처분 안건 against 트리거

### 6.2 공시별 영향

| 공시 | 신법 전 | 신법 후 |
|------|--------|--------|
| 자기주식취득결정 | `aq_pp` 모호 가능 | **"소각" 명시 필수** (commitment) |
| 자기주식처분결정 | 자유 (자사주 마법) | **합리적 사유 + 외부 검증** (제3자 매각 사실상 불가) |
| 자기주식소각결정 | 자율 | **준의무화** — 빈도 급증 (50건/년 → 200건+ 예상) |
| 자기주식 신탁계약 | 자유 | 체결 단계에서 "소각" 명시 권장, 신탁 매입분도 1년 의무 |
| 분쟁 중 처분 (한진칼-KCGI 패턴) | 빈번 | **against 절대** |

### 6.3 임팩트 추정

| 시점 | 변화 |
|------|------|
| 2026.03 ~ 2026.09 | 신법 시행 직후 — 취득 결정 일시 감소 (관망), 일부 미리 소각 |
| 2026.09 ~ 2027.09 | 기존 보유분 처리 의무 본격화 — 소각 빈도 급증, 처분 빈도 급감 |
| 2027.09 ~ | 자사주 비중 글로벌 표준 (1-2%) 정착, 자사주 마법 사실상 소멸 |

### 6.4 OPM 정책 ([[open-proxy-guideline]] 2.7)

- **for**: 자기주식 취득 + 1년 내 소각 commitment 명시
- **for**: 합리적 사유 + 처분 결과 주총 보고
- **against**: 자사주 처분 + 1년 내 소각 계획 없음
- **against 절대**: 분쟁 중 자사주 처분
- **review**: 경영권 보호 목적 취득 (사실상 against)

## 9. 거버넌스 분석 시나리오

### 시나리오 1: 적극 주주환원 모범 (KT&G)
1. 자기주식취득결정 (`aq_pp` = "주주가치 제고 및 소각") → for
2. 1년 내 자기주식소각결정 → for
3. 동시에 현금배당결정 (DPS 안정 / 증가) → for
- → 종합: 적극 주주환원 우수 회사

### 시나리오 2: 자사주 마법 의심 (한진칼-KCGI 패턴)
1. 자기주식취득결정 (`aq_pp` = "주가 안정화") → review
2. 1년 후 자기주식처분결정 (`dpptncmp_cmpnm` = 우호 세력) → **against 절대**
3. 분쟁 중 (소송, 5% 보고서 변동, 위임장 권유) → **against 절대**
- → 종합: 경영권 방어 자사주 마법 시도

### 시나리오 3: 감액배당 (배당 재원 부족)
1. 영업현금흐름 음수 + 이익잉여금 부족
2. 자본준비금 감소 결정 → 정관변경 + 주총 특별결의 안건 → review
3. 감액배당으로 현금배당결정 → review/against (지속가능성 의문)
- → 종합: 부실 신호 + 일회성 환원

### 시나리오 4: 분기배당 정착 (POSCO홀딩스, 삼성전자)
