# 회사분할합병결정

## 개요

회사를 분할한 뒤 그 분할된 부분을 다른 회사와 곧바로 합병하는 결정. **분할 + 합병 결합 도구**. 사업부 분리 후 외부 회사로 이전, 또는 그룹 내 사업 재배치에 사용.

상법상 분할합병은 단순 분할이나 단순 합병보다 절차가 무겁고 외부평가·매수청구권 모두 적용된다.

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=B (주요사항보고)
- **DART API**: `cmpDvmgDecsn.json` (DS005 주요사항보고서)
- **report_nm 패턴**: `주요사항보고서(회사분할합병결정)`, `회사분할합병결정`
- **의무 여부**: 의무
- **공시 주기**: 수시 (이사회 결의 익영업일까지)

## 전체 문서 구조 (트리)

기본 구조는 [[회사분할결정]]과 [[회사합병결정]]을 결합한 형태. 응답 필드도 두 공시의 필드를 합친 superset.

```
주요사항보고서(회사분할합병결정)
│
├── 1. 분할합병 형태 및 방법
│   ├── 분할형태 (인적 / 물적)                  ← ex_sm_r
│   ├── 분할방법                               ← dv_mth
│   └── 합병상대방 (분할되는 사업이 합병될 회사)  ← mgptncmp_cmpnm
│
├── 2. 분할비율 및 합병비율
│   ├── 분할비율                               ← dv_rt
│   ├── 합병비율                               ← mg_rt
│   └── 산정근거                               ← mg_rt_bs
│
├── 3. 분할대상 사업                            ← dv_trfbsnprt_cn
│
├── 4. 합병 상대회사 (합병 흡수자)
│   ├── 회사명                                 ← mgptncmp_cmpnm
│   ├── 사업내용                               ← mgptncmp_mbsn
│   ├── 본 회사와의 관계                        ← mgptncmp_rl_cmpn
│   └── 재무                                   ← rbsnfdtl_*
│
├── 5. 존속회사 (분할 후 남는 회사)             ← atdv_excmp_*
│
├── 6. 외부평가                                 ← exevl_*
│
├── 7. 주식매수청구권                           ← aprskh_*
│
├── 8. 일정                                    ← gmtsck_prd / dvdt 등
│
├── 9. 풋옵션                                   ← popt_ctr_*
│
└── 10. 이사회 결의                             ← bddd / od_a_at_t / od_a_at_b
```

## API 필드 vs 원문 대응

[[회사합병결정]]과 [[회사분할결정]]의 필드를 통합 사용. 추가 검증 시 양 페이지 참조.

| API 필드 | 의미 | 비고 |
|----------|------|------|
| `bddd` | 이사회 결의일 | |
| `ex_sm_r` | 분할형태 (인적/물적) | |
| `dv_rt` | 분할비율 | |
| `mg_rt` | 합병비율 (분할된 사업 ↔ 흡수회사) | |
| `dv_trfbsnprt_cn` | 분할대상 사업 | |
| `mgptncmp_cmpnm` | 합병 상대회사 (흡수자) | |
| `mgptncmp_rl_cmpn` | 합병 상대회사 관계 | 자회사면 그룹 내 재배치 |
| `rbsnfdtl_*` | 합병 상대회사 재무 | |
| `atdv_excmp_*` | 존속회사 정보 | |
| `aprskh_plnprc` | 매수청구권 가격 | |
| `popt_ctr_atn` | 풋옵션 약정 여부 | |

> **주의**: 응답 데이터 표본이 적어 필드 일부의 정확한 사용처는 추가 검증 필요. 실제 케이스 발생 시 응답 구조 재 probe 권장.

## OPM에서의 활용

- **`corporate_restructuring(scope="split")`**: 분할결정과 통합 표시. event_label에 "회사분할합병결정"으로 구분.
- **`corporate_restructuring(scope="summary")`**: 4종 timeline에 통합.

## 거버넌스 분석 포인트

1. **분할 + 합병 동시 실행**: 사업부 외부 이전 → 그룹 외 가치 이전 가능성. 합병 상대회사가 특수관계인이면 일감몰아주기 의심.
2. **인적분할합병**: 분할 후 합병 대가로 합병 흡수자의 신주를 분할회사 주주에게 배정 → 주주 구성 변경.
3. **존속회사 가치 변화**: 분할대상 사업의 매출 비중에 따라 존속회사 잔여가치 평가 필요.
