# 현금배당결정

이사회에서 현금(또는 현물) 배당을 결정한 후 제출하는 거래소 수시공시. 배당 이벤트의 핵심 정보(DPS, 배당기준일, 지급일, 시가배당률, 총액, 차등 여부 등)가 한 번에 담기는 필수 공시.

## 개요

| 항목 | 값 |
|------|-----|
| 공시 의무 회사 | 상장사 전체 (KOSPI / KOSDAQ / KONEX) |
| 제출처 | KRX-KIND (거래소 공시), DART는 `pblntf_ty="I"`로 수신 |
| 시기 | 이사회결의일 다음 영업일까지 |
| 의무 여부 | 의무 (자본시장법 §161 ① 7호 + 거래소 공시규정) |
| 공시 주기 | 수시 (배당 결정 시) |
| 한 회사 / 연 | 결산배당 1건 + 분기·중간배당 추가 (분기배당 기업은 연 4건) |

## 핵심 데이터 항목

| 필드 (한글) | 형식 | 예시 | 설명 |
|---|---|---|---|
| 배당구분 | enum | 결산배당 / 분기배당 / 중간배당 / 특별배당 | 분기배당과 중간배당은 동일 의미로 혼용 표기 |
| 배당종류 | enum | 현금배당 / 현물배당 | 현물배당 시 자산 상세 기재 |
| 1주당 배당금 (보통주) | int (원) | 1,500 | DPS 핵심 필드 |
| 1주당 배당금 (종류주식) | int (원) | 1,550 | 우선주별 배열 — 별도 테이블에 기재 |
| 차등배당 여부 | bool | 미해당 / 해당 | 대주주 차등배당 여부 |
| 시가배당률 (보통주) | float (%) | 2.34 | 이사회결의일 직전 1주일 평균 종가 기준 |
| 시가배당률 (종류주식) | float (%) | 2.41 | 우선주 별도 |
| 배당금 총액 | int (원) | 9,739,849,000,000 | 보통주 + 종류주식 + 우선주 합산 |
| 배당기준일 | date | 2025-12-31 | 권리주주 확정일 (결산배당은 통상 결산일) |
| 배당금 지급 예정일 | date / "-" | 2026-04-19 | 결산배당은 주총 후 1개월 이내, 미정 시 "-" |
| 주주총회 개최여부 | enum | 개최 / 미개최 | 결산배당=개최, 분기배당=미개최 |
| 주주총회 예정일자 | date / "-" | 2026-03-19 | 결산배당만 |
| 이사회결의일 | date | 2026-01-29 | 결정일 |
| 사외이사 참석여부 | str | 참석(명) 5 / 불참(명) 0 | |
| 감사 참석여부 | str | - / 참석 | 사외이사 아닌 감사위원 |
| 기타 투자판단 중요사항 | text | 자유기재 | 차등배당 사유, 정관 조문 등 |

## DART API 매핑

- **API**: 별도 구조화 API 없음. `list.json` (`pblntf_ty="I"` + `corp_code` + 기간)으로 검색
- **report_nm 패턴**: `현금ㆍ현물배당결정`, `[기재정정]현금ㆍ현물배당결정`
- **본문 파싱**: `evidence(rcept_no)` → DART 뷰어 HTML/PDF에서 위 트리 항목 직접 추출
- **사업보고서 alotMatter**: 연간 합산 DPS 별도 — 회차별 vs 연간 합산 구분 필수

## OPM 활용

- **`div_search(ticker)`** — 배당 관련 공시 검색
- **`div_detail(ticker, bsns_year, reprt_code)`** — alotMatter 기반 + 회차별 공시 보강
- **`div_history(ticker, years)`** — 연도별 추이 (분기배당은 4건 합산)
- **연 1회 vs 분기배당 구분**: 한 기업이 같은 연도 내 다중 공시 → 회차별로 식별 필요
- **시가배당률 계산 기준 주의**: 이사회결의일 직전 1주일 평균 (배당기준일 기준 아님)
- **자회사판 중복 제거**: `(자회사의 주요경영사항)` 버전은 모회사가 자회사 배당을 대신 공시 — KOSPI 200 모회사 시각에서는 시장 배당 이벤트가 아닌 모회사 수입

## 거버넌스 분석 포인트

- **DPS 추세**: 3년 추이 + [[배당성향]] (지배주주 귀속 당기순이익 대비)
- **시가배당률**: 동종업계 평균 대비 과소 / 과다
- **차등배당**: 대주주 사익 추구 신호 가능 — `차등배당 여부 = 해당` 시 review
- **현물배당**: 흔치 않음 — 자산 가치평가 적정성 별도 검증 필요
