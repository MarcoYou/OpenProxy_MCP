## 개요

사업연도 종료 후 90일 이내 제출하는 종합 보고서. 재무제표, [[지분구조]], 배당 정보를 모두 포함하는 가장 상세한 정기 공시.

## 소스

- **제출처**: [[DART-OpenAPI]] (reprt_code=11011)
- **의무 여부**: 의무
- **공시 주기**: 정기 (연 1회, 사업연도 종료 후 90일 이내)

## 주요 내용

- 연결/별도 재무제표 전체
- [[최대주주]] + [[특수관계인]] 현황
- [[소액주주]] 현황
- [[자사주]] 보유/변동 현황
- 배당 상세 (DPS, [[배당성향]], [[시가배당률]])
- 임원 보수 현황

## 전체 문서 구조 (트리)

```
사업보고서
│
├── 제1부 회사의 개요
│   ├── 회사의 법적·상업적 연혁
│   ├── 자본금 변동 현황
│   └── 주식의 총수 현황  ← own_total 소스 (stockTotqySttus API)
│
├── 제2부 사업의 내용
│   ├── 사업 부문별 현황 및 시장 점유율
│   └── 주요 제품 및 서비스
│
├── 제3부 재무에 관한 사항
│   ├── 연결재무제표 (지배주주귀속 순이익 포함)  ← 실적 분석 기준
│   │   ├── 연결 재무상태표
│   │   ├── 연결 손익계산서
│   │   └── 연결 현금흐름표
│   └── 별도재무제표
│       ├── 별도 재무상태표
│       └── 별도 손익계산서
│
├── 제4부 감사인의 감사의견
│   └── 적정/한정/부적정/의견거절
│
└── 제5부 기타
    ├── 임원 및 직원 현황
    │   └── 등기임원 보수 현황 (개인별 또는 총액)  ← agm_compensation 교차 확인
    ├── 최대주주 및 특수관계인 현황  ← own_major 소스 (hyslrSttus API)
    │   └── 성명, 관계, 보유주식수, 지분율, 변동내역
    ├── 주식의 총수 현황  ← own_total 소스 (stockTotqySttus API)
    │   └── 발행주식 총수, 자기주식, 유통주식
    ├── 자기주식 취득 및 처분 현황  ← own_treasury 소스 (tesstkAcqsDspsSttus API)
    │   └── 취득/처분 일자, 수량, 단가, 방법
    └── 배당에 관한 사항  ← div_* 소스 (alotMatter API)
        └── 연간 DPS (보통주/우선주), 배당성향, 시가배당률
```

API: DART `reprt_code=11011` (사업보고서), DS002 정기보고서 다수 endpoint 사용

## API 필드 vs 원문 대응

| API / Endpoint | 원문 위치 | 실제 의미 | 주의사항 |
|----------------|-----------|-----------|----------|
| `hyslrSttus` | 제5부 > 최대주주 현황 | 최대주주 + 특수관계인 지분 현황 | own_major 소스 |
| `stockTotqySttus` | 제5부 > 주식 총수 현황 | 발행주식 총수, 자기주식, 유통주식 | own_total 소스 |
| `tesstkAcqsDspsSttus` | 제5부 > 자기주식 취득/처분 | 자기주식 변동 내역 | own_treasury 소스 |
| `alotMatter` | 제5부 > 배당에 관한 사항 | 연간 합산 DPS, 배당성향, 시가배당률 | **회차별 DPS 아님** — 현금배당결정 공시와 구분 |
| `reprt_code` | 표지 | 보고서 유형 코드 | `11011` = 사업보고서 |
| `bsns_year` | 표지 | 사업연도 | |

**연결 지배주주귀속 순이익은 별도 API 없음.**
→ `get_document()` 또는 재무제표 API (`fnlttSinglAcntAll`)로 `연결_지배주주귀속` 항목 파싱 필요.

## OPM에서의 활용

지분 tool과 배당 tool의 핵심 소스:
- own_major: hyslrSttus API로 [[최대주주]] 현황 조회
- own_total: stockTotqySttus API로 발행주식/[[자사주]]/유통주식 조회
- own_treasury: tesstkAcqsDspsSttus API로 [[자사주]] 변동 조회
- div_detail: alotMatter API로 [[배당성향]], [[배당수익률]] 조회

[[own-tool-rule]]에서 사업보고서를 데이터 소스 우선순위 1위로 지정.
