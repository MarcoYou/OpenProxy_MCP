# 신탁계약 해지 결과보고서

신탁계약 해지결정 후 **신탁 사이클 종료 결과**를 보고하는 공시.

## 개요

| 항목 | 값 |
|------|-----|
| 트리거 | 신탁계약 해지 |
| 기한 | 해지일로부터 **5일 이내** |
| 의무 여부 | 의무 |
| 공시 주체 | 상장법인 |
| 1차 채널 | 금융위(DART) |
| API | **별도 구조화 API 없음** — `list.json` + 키워드 |
| report_nm 패턴 | `신탁계약해지결과보고서` 또는 `신탁계약 해지 결과보고서` |

## 핵심 데이터

### list.json 메타
| 필드 | 의미 |
|---|---|
| `report_nm` | 신탁계약 해지 결과보고서 |
| `rcept_no` / `rcept_dt` | 접수번호 / 일자 |

### 본문 항목 (예: 대한조선 rcept_no=20260429000799)
| 항목 | 의미 |
|---|---|
| 신탁계약 체결일 | 원 사이클 시작 |
| 해지일 | 신탁 종료 |
| 총 취득실적 | 사이클 전체 합계 (수량/금액) |
| 미달 사유 | 취득예정금액 미달 시 |
| 해지 후 자기주식 보유현황 | 직접보유(A) + 신탁(B=0) 합계 |
| 잔여재산 처리 | 현금 환수 / 재신탁 등 |

## 사이클 위치

```
신탁체결 → 상황보고서 (분기 반복) → 신탁해지결정 → [해지결과보고서] → 잔고 (사업보고서)
```

## OPM 통합

- `treasury_share` phase=`execution`, 사이클 종료 marker
- 사이클별 grouping에서 마지막 row
- 총 취득실적 ↔ 신탁계약금액 비교로 신탁 효율 측정
