# 자기주식 신탁계약 체결/해지결정

회사가 자기주식을 직접 매입하지 않고 **신탁사(증권사 등)에 위임**하여 일정 기간 분산 매입하는 결정 공시 + 그 신탁계약을 만기 또는 중도 해지하는 결정 공시. 직접 취득 대비 시장 영향 완화 + 매입 부담 분산. **체결과 해지 2종 공시 별개**.

## 개요

| 항목 | 값 |
|------|-----|
| 공시 의무 회사 | 상장사 전체 (자사주 신탁계약 결정 시) |
| 제출처 | DART (주요사항보고서 — DS005 그룹) |
| 시기 | 이사회결의일 다음 영업일까지 |
| 의무 여부 | 의무 |
| 공시 주기 | 수시 |
| API | `tsstkAqTrctrCnsDecsn.json` (체결) + `tsstkAqTrctrCcDecsn.json` (해지) |
| pblntf_ty | B (주요사항보고서) |
| 신탁 유형 | 자기주식 취득 신탁 (대부분), 매입형/처분형 구분 가능 |

## 1. 자기주식 취득 신탁계약 체결결정 (tsstkAqTrctrCnsDecsn)

회사가 신탁사에 자사주 매입을 위임하는 계약을 체결한 결정 공시.

### 핵심 데이터 항목

| API 필드 | 한글 | 형식 | 예시 | 설명 |
|---|---|---|---|---|
| `bddd` / `ctr_cns_dd` | 이사회결의일 | date | 2026-04-15 | |
| `cs_iv_bk` | 신탁 수탁자 | str | 한국투자증권, 미래에셋증권 등 | 핵심 |
| `ctr_prc` (또는 `ctr_prc_am`) | 계약 금액 | int (원) | 50,000,000,000 | 신탁 한도 |
| `ctr_cns_prd_bgd` (또는 `ctr_prd_bgd`) | 계약 기간 시작 | date | 2026-04-16 | |
| `ctr_cns_prd_edd` (또는 `ctr_prd_edd`) | 계약 기간 종료 | date | 2027-04-15 | 보통 6개월~1년 |
| `ctr_pp` | 계약 목적 | text | 주주가치 제고 등 | **2026.03 신법 — "소각" 명시 여부** |
| `od_a_at_t` / `od_a_at_b` | 사외이사 참석/불참 | int | 5 / 0 | |

## 2. 자기주식 취득 신탁계약 해지결정 (tsstkAqTrctrCcDecsn)

진행 중인 자사주 신탁계약을 만기 도달 또는 중도 해지하는 결정 공시. **신탁 기간 중 실제 매입 결과**(`aq_stk_ostk`)를 확인 가능.

### 핵심 데이터 항목

| API 필드 | 한글 | 형식 | 예시 | 설명 |
|---|---|---|---|---|
| `bddd` / `ctr_cc_dd` | 이사회결의일 | date | 2027-04-10 | |
| `cs_iv_bk` | 신탁 수탁자 | str | 한국투자증권 | |
| `ctcccr_dt` | 해지일 | date | 2027-04-15 | |
| `ctcccr_rs` | 해지 사유 | text | 만기 도래, 중도 해지 | |
| `aq_stk_ostk` | 신탁 기간 중 취득한 보통주 수 | int | 950,000 | **실제 매입 결과** — 한도 대비 비율 검증 |
| `aq_stk_estk` | 신탁 기간 중 취득한 종류주식 수 | int | 0 | |
| `aq_prc_ostk` | 실제 취득 금액 | int (원) | 47,500,000,000 | 한도 대비 비율 |
| `od_a_at_t` / `od_a_at_b` | 사외이사 참석/불참 | int | 5 / 0 | |

### 전체 문서 구조 (트리)

```
[회사명]/주요사항보고서(자기주식취득신탁계약해지결정)/...

자기주식 취득 신탁계약 해지 결정
│
├── 1. 신탁계약 해지 사유      만기 도달 / 중도 해지
├── 2. 해지일                 YYYY-MM-DD
├── 3. 계약 체결기관 (수탁자)   증권사명
├── 4. 신탁 기간 중 취득 결과
│   ├── 보통주 수량 (주)        XXX,XXX
│   ├── 종류주식 수량 (주)      -
│   └── 실제 취득 금액 (원)     XXX,XXX,XXX,XXX  (한도 대비 X.X%)
├── 5. 자기주식 처분 후속 계획
├── 6. 이사회결의일             YYYY-MM-DD
```

| 신탁계약 해지 | `tsstkAqTrctrCcDecsn.json` | DS005 | `주요사항보고서(자기주식취득신탁계약해지결정)` |

- **응답 구조**: `list[]` 배열

## OPM 활용

- **`treasury_share(scope="acquisition")`** — 신탁 체결 포함 (직접 취득 + 신탁 체결 통합)
- **`treasury_share(scope="disposal")`** — 신탁 해지 포함
- **`treasury_share(scope="events")`** — 5종 통합 timeline
- **실집행 비율 검증**: 체결 시 한도 vs 해지 시 실제 매입 수량 비교 — 공약 이행도 평가

## 거버넌스 분석 포인트

### 신탁 vs 직접 취득
| 항목 | 직접 취득 | 신탁 |
|------|---------|------|
| 시장 영향 | 직접 (단기) | 분산 (완화) |
| 매입 시점 통제 | 회사 직접 | 신탁사 재량 |
| 매입 한도 vs 실제 | 통상 100% 가까이 | 종종 50-90% (시장 상황 따라) |
| 비용 | 거래 수수료만 | + 신탁 수수료 |

### 실집행 비율 (`aq_stk_ostk` / 한도)
- **90% 이상**: 적극 매입 (시장 상황 우호)
- **50-90%**: 일부 매입
- **50% 미만**: 형식적 신탁 — 주주환원 의지 약함 신호

### 2026.03 신법 영향 (의무소각)
- **신탁 매입 → 회사 보유 → 1년 내 소각 의무 발생**
- **체결 단계에서 `ctr_pp`에 "소각" 명시 권장** — 미명시 시 OPM against
- **신탁 해지 후 매입한 주식의 소각 일정 명시**: 후속 [[자기주식소각결정]] 공시 필수

### 신탁 기간 만료 후 처분 (`자사주 마법` 변형)
- 신탁 매입 후 회사 보유 → 처분 시 `dpptncmp_cmpnm` 채워짐 → 우호 지분 가능
- 신법으로 차단되었으나 우회 사례 모니터링 필요
