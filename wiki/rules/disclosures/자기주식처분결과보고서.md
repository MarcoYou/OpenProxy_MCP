# 자기주식처분결과보고서

자기주식 처분결정 후 **실제 처분 완료 결과**를 보고하는 공시.

## 개요

| 항목 | 값 |
|------|-----|
| 트리거 | 처분기간 종료 |
| 기한 | 종료일 다음날부터 **5일 이내** (자본시장법 시행령 §176조의5 ③) |
| 의무 여부 | 의무 |
| 공시 주체 | 상장법인 |
| 1차 채널 | 금융위(DART) |
| API | **별도 구조화 API 없음** — `list.json` + 키워드 |
| report_nm 패턴 | `자기주식처분결과보고서` |
| 본문 파싱 | 일자별 실제 처분 raw + 합계 + 상대방 |

## 핵심 데이터

### list.json 메타
| 필드 | 의미 |
|---|---|
| `corp_code` | 기업코드 |
| `report_nm` | "자기주식처분결과보고서" |
| `rcept_no` | 접수번호 |
| `rcept_dt` | 접수일자 |

### 본문 항목 (예: 삼성전자 rcept_no=20260325001270)
| 항목 | 의미 |
|---|---|
| 주요사항보고서 제출일 | 원래 처분결정 공시일 |
| 처분기간 | 매도 시작일 ~ 종료일 |
| 일자별 처분수량 / 단가 / 상대방 | 일별 raw 표 |
| 처분 합계 | 총 주식수, 총 금액 |
| 미달 사유 | 처분예정금액 미달 시 |

## 결정 ↔ 결과 매칭

| 사이클 | 매칭 키 |
|---|---|
| 처분결정 (`tsstkDpDecsn`) → 처분결과보고서 | "주요사항보고서 제출일" 동일 |

## OPM 통합

- `treasury_share` tool phase=`execution` 이벤트로 추가
- 처분 상대방 (3자 배정 등) 명시 시 ownership 변동 cross-ref
