# 회사합병결정

## 개요

이사회에서 다른 회사와의 합병을 결정하면 제출하는 의무 공시. 흡수합병/신설합병, 소규모합병/일반합병 구분과 함께 합병비율·상대회사 재무·매수청구권·외부평가 의견 등 거버넌스 핵심 정보가 담긴다.

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=B (주요사항보고)
- **DART API**: `cmpMgDecsn.json` (DS005 주요사항보고서)
- **report_nm 패턴**: `주요사항보고서(회사합병결정)`, `[기재정정]주요사항보고서(회사합병결정)`
- **의무 여부**: 의무
- **공시 주기**: 수시 (이사회 결의 익영업일까지)

## 전체 문서 구조 (트리)

```

│
├── 1. 합병 방법, 형태, 목적
│   ├── 합병방법 (흡수합병 / 신설합병)         ← mg_mth
│   ├── 합병형태 (소규모 / 일반 / 간이)         ← mg_stn

│
├── 2. 합병비율 및 산정근거
│   ├── 합병비율 (예: A:B = 1 : 0.5234)       ← mg_rt
│   └── 비율 산정근거 (자본시장법 §165조의5)    ← mg_rt_bs
│
├── 3. 합병신주 발행
│   ├── 보통주 신주 수                         ← mgnstk_ostk_cnt
│   └── 종류주식 신주 수                       ← mgnstk_cstk_cnt
│
├── 4. 합병 상대회사
│   ├── 회사명                                 ← mgptncmp_cmpnm

│   ├── 본 회사와의 관계                        ← mgptncmp_rl_cmpn (자회사/관계회사 등)
│   ├── 재무현황                               ← rbsnfdtl_*
│   │   ├── 자산총계 (tast)
│   │   ├── 부채총계 (tdbt)

│
├── 5. 외부평가 (자본시장법상 의무)
│   ├── 평가 여부                              ← exevl_atn
│   ├── 평가기관                               ← exevl_intn

├── 6. 주식매수청구권
│   ├── 행사가격                               ← aprskh_plnprc

├── 8. 우회상장 해당 여부                      ← bdlst_atn / otcpr_bdlst_sf_atn
```

## API 필드 vs 원문 대응

| API 필드 | 원문 위치 | 의미 | 주의사항 |
|----------|-----------|------|----------|
| `bddd` | 11. 이사회결의 | 이사회 결의일 | 합병계약일과 다를 수 있음 |
| `mg_stn` | 1. 형태 | 소규모합병 / 일반합병 / 간이합병 | 소규모합병은 주총 생략 가능 |
| `mg_mth` | 1. 방법 | 흡수합병 (A가 B 흡수) / 신설합병 (A+B → C) | 신설합병이면 nmgcmp_cmpnm 채워짐 |
| `mg_pp` | 1. 목적 | 합병목적 자유기재 | "효율적 의사결정 체제" 등 정형 문구 빈번 |
| `mg_rt` | 2. 비율 | 합병비율 | "A : B = 1 : 0.X" 포맷, 텍스트 파싱 필요 |
| `mg_rt_bs` | 2. 근거 | 자본시장법 시행령 제176조의5 인용 | 산정 방법론 (시가/본질가치 등) |
| `mgnstk_ostk_cnt` | 3. 신주 | 보통주 합병신주 수 | 0이면 무상합병 (소규모합병 가능성) |
| `mgptncmp_cmpnm` | 4. 상대회사 | 합병 상대회사명 | (주) 표기 변형 주의 |
| `mgptncmp_rl_cmpn` | 4. 관계 | 자회사/관계회사/제3자 | "자회사" 면 100% 흡수합병 패턴 |
| `rbsnfdtl_tast/tdbt/teqt` | 4. 재무 | 상대회사 자산/부채/자본 | 직전 사업연도 기준, 단위 원 |
| `rbsnfdtl_sl/nic` | 4. 재무 | 상대회사 매출/순이익 | 적자(-) 회사 합병 시 음수 |
| `exevl_intn` | 5. 외부평가 | 평가기관 (회계법인) | 자율 평가 vs 의무 평가 구분 |
| `exevl_op` | 5. 외부평가 의견 | 합병가액/비율 적정성 의견 | 평가 핵심, 200자 이내 요약 권장 |
| `aprskh_plnprc` | 6. 매수청구권 | 행사가격 | 단위 원, 시가 평균 기준 산정 |
| `popt_ctr_atn` | 7. 풋옵션 | 풋옵션 약정 여부 (예/아니오) | "예" 면 재무위험 신호 |
| `bdlst_atn` | 8. 우회상장 | 우회상장 해당 여부 | 비상장 회사 합병 시 검토 대상 |

## OPM에서의 활용

- **`corporate_restructuring(scope="merger")`**: 합병결정 카드 형태로 표시. 비율/상대방/매수청구권/외부평가/풋옵션을 한 화면에 정렬.
- **`corporate_restructuring(scope="summary")`**: 다른 재편 사건과 함께 timeline에 통합. 상대회사명 + 합병비율 노출.
- **`evidence`**: rcept_no로 원문 viewer URL 생성.

## 거버넌스 분석 포인트

1. **합병비율 적정성**: 외부평가 의견과 산정근거 (`mg_rt_bs`)의 합리성. 자회사 흡수합병에서 비율이 모회사에 유리하게 책정되는 경우 다수.
2. **소규모합병**: 주총 생략 가능 → 소수주주 의결권 행사 기회 박탈. `mg_stn=소규모` 케이스 주의.
3. **풋옵션 약정**: `popt_ctr_atn=예` 면 합병 후 일방적 매도 권리 부여. 우호 인수자에게 유리한 조건.
4. **상대회사 적자**: `rbsnfdtl_nic` 음수 합병은 모회사 가치 희석. 부실 자회사 흡수 가능성.
5. **매수청구권 가격**: 시가 대비 할인/할증 정도 — 반대주주 보호 수위.
