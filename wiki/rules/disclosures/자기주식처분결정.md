# 자기주식처분결정

보유 중인 자기주식을 처분(매각, 임직원 부여, 합병 대가, 제3자 배정 등)하기로 결정하는 공시. **자사주 마법(처분 → 우호 지분 확보)**의 직접 도구. 2026.03 신법 시행으로 처분 사유 + 1년 내 소각 commitment 없는 처분은 OPM에서 against 판정.

## 개요

| 항목 | 값 |
|------|-----|
| 공시 의무 회사 | 상장사 전체 (자기주식 처분 결정 시) |
| 제출처 | DART (주요사항보고서 — DS005 그룹) |
| 시기 | 이사회결의일 다음 영업일까지 |
| 의무 여부 | 의무 (자본시장법 §161) |
| 공시 주기 | 수시 |
| API | `tsstkDpDecsn.json` (DART OpenAPI 구조화 제공) |
| pblntf_ty | B (주요사항보고서) |

## DART API 매핑

- **endpoint**: `https://opendart.fss.or.kr/api/tsstkDpDecsn.json`
- **그룹**: DS005 (자기주식 결정)
- **파라미터**: `corp_code`, `bgn_de`, `end_de`
- **응답 구조**: `list[]` 배열 — 위 필드 모두 포함
- **report_nm 패턴**: `주요사항보고서(자기주식처분결정)`, `[기재정정]` 동반 가능

## OPM 활용

- **`treasury_share(scope="disposal")`** — 처분결정만 조회
- **`treasury_share(scope="events")`** — 5종 통합 timeline
- **자사주 마법 자동 검출**: `dpptncmp_cmpnm` 필드 채워짐 + `dp_pp`에 "임직원/우리사주" 외 사유 → 우호 지분 확보 우려 플래그
- **`screen_events(event_type="treasury_dispose")`** — 시장 전체 검색

## 거버넌스 분석 포인트

### 처분 목적별 분류 (`dp_pp`)

| 목적 | 분석 의미 | OPM guideline default |
|------|---------|----------------------|
| 임직원 상여 / 우리사주조합 | 보상 정책 일부 (비교적 중립) | review |
| 스톡옵션 행사 대응 | 보수 정책 | review |
| 합병 대가 | M&A 거래의 일부 | review (M&A 별도 검증) |
| 제3자 매각 (`dpptncmp_cmpnm` 채워짐) | **우호 지분 확보 (자사주 마법)** | against |
| 자금 조달 | 재무 악화 신호 | against |
| 새 사업 / 신규 투자 | 합리적 사유 검증 필요 | review/against |

### 자사주 마법 검증
- `dpptncmp_cmpnm`이 채워졌는가? → YES면 제3자 매각
- 처분 상대가 특수관계인 또는 우호 세력인가? → YES면 자사주 마법
- 분쟁 중 (소송, 5% 보고서 변동, 위임장 권유) 처분인가? → 한진칼-KCGI 패턴

### 2026.03 신법 영향
- **신법 핵심**: 자기주식은 1년 내 소각이 원칙 — 처분은 예외
- **OPM guideline (against)**: 자사주 처분 + 1년 내 소각 계획 없음
- **합리적 사유 + 외부 검증**: M&A 대가, 우리사주, 임직원 상여 → review
- **분쟁 중 처분**: against 절대

### 처분 방법 (`dp_mth`)
- **장내매도**: 시장 가격, 매수자 불특정 → 비교적 중립
- **시간외매도**: 가격 불투명, 특정 매수자 가능
- **제3자 배정**: 가장 논쟁적 — 우호 지분 만들기
