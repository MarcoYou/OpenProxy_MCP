# 주주환원

## 개념

주주환원 = 회사가 주주에게 돌려주는 가치. 두 가지 정의가 혼용된다 — 어느 쪽인지 항상 명시할 것.

### CSR (Cash Shareholder Return) — 한국 시장 정의

```
CSR = (배당총액 + 자사주 매입 금액) / 지배주주 당기순이익 × 100
```

- 회사가 주주에게 돌려준 **현금** 관점 (회사 회계 입장)
- 분자에는 자사주 **매입(acquire)** 사용. 소각(retire)은 회계 정리 단계로 매입 시점에 이미 현금이 나갔으므로 분자에 사용하면 이중 계산.
- 분모는 한국 표준대로 연결 지배주주 당기순이익.
- OPM tool: `dividend(scope="cash_shareholder_return")`

### TSR (Total Shareholder Return) — 글로벌 정의

```
TSR = (P_end - P_start + Dividends) / P_start × 100
```

- **주식 투자자 관점**의 1주 수익률 (capital gain + income)
- 주가 변동 + 1주당 배당금 합계
- 음수 이익(적자) 회사도 산출 가능
- OPM tool: `dividend(scope="total_shareholder_return")`

## CSR vs TSR

같은 "주주환원"이라는 단어를 쓰지만 다른 지표. 한국 자본시장 분석에서는 보통 CSR을 의미하고, Anglo-Saxon 분석(MSCI, ISS, Glass Lewis)에서는 TSR을 의미한다.

| 항목 | CSR | TSR |
|---|---|---|
| 분자 | 배당 + 자사주 매입 | 주가 변동 + 배당 |
| 분모 | 지배주주 당기순이익 | 시작 주가 (P_start) |
| 단위 | 회사 전체 (절대 KRW) | 1주 |
| 관점 | 회사 회계 | 투자자 수익률 |
| 적자 회사 | 비율 계산 안 됨 | 산출 가능 |

## OPM에서의 처리

- `dividend(scope="cash_shareholder_return")` — CSR (한국식)
- `dividend(scope="total_shareholder_return")` — TSR (글로벌)
- `treasury_share(scope="acquisition")` — 분자 매입 source
- `treasury_share(scope="cancelation")` — 회계 정리 단계 (CSR 분자 X)
- 둘 다 호출하면 회사 정책 + 주가 양쪽 관점 비교 가능
