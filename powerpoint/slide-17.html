related-party / intercompany transactions

related_party_transaction

타법인주식 거래와 단일공급계약을 묶어 내부거래 성격의 이벤트를 보여준다.

Role

일감몰아주기나 이해상충성 거래를 빠르게 훑는 related-party tab.

Inputs

  • company
  • lookback window
  • default summary scope

Outputs

  • transaction event summary
  • counterparty / contract highlights
  • related risk context

Sources

  • 타법인주식 공시
  • 단일공급계약 공시
  • keyword filtering

Workflow

  • 관련 거래 공시 탐색
  • 거래 유형 정규화
  • risk-oriented summary 반환

Notes

정밀 법률판단보다는 내부거래 탐색용 screening tab에 가깝다.
