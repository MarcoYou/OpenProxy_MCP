evidence lookup / rcept_no to URLs

evidence

rcept_no를 받아 공시일, source, viewer URL 등 근거 확인에 필요한 최소 정보를 반환한다.

Role

analysis 결과를 실제 공시 원문과 연결해 주는 evidence utility tool.

Inputs

  • rcept_no or evidence refs
  • single or list mode
  • no extra scope

Outputs

  • viewer URL / source URL
  • filed date / metadata
  • traceability block

Sources

  • rcept_no string processing
  • viewer URL construction
  • stored metadata fields

Workflow

  • evidence ref 해석
  • URL / metadata 조립
  • trace block 반환

Notes

action payload의 evidence_rcept_no를 실제 공시 확인 단계로 이어주는 bridge다.
