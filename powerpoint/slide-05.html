company

모든 질문 흐름의 공통 입구다. 회사명, ticker, corp_code를 canonical entity로 정리하고 최근 공시 인덱스를 붙여 downstream tool 호출 기반을 만든다.

Role

기업 식별, 업종/시장/기본 프로필, 최근 공시 인덱스를 한 번에 정리하는 discovery tool.

Inputs

  • company / ticker / corp_code
  • scope = summary | filings 등
  • lookback / page count

Outputs

  • canonical entity info
  • induty_code + KSIC 업종명
  • recent filings index

Sources

  • DART corp code / company endpoint
  • DART list.json
  • 로컬 KSIC 매핑

Workflow

  • 회사명 정규화
  • corp_code 매칭
  • 프로필 + filings 인덱스 조립

Notes

모든 data/action tool의 공통 진입점이다. NAVER 제거 후 DART + KSIC만 사용한다.
