Slide 03

Source → Service → Tool → Client

전체 시스템은 공시 source를 parser/service layer에서 정리하고, 이를 16개 MCP tool로 노출한 뒤, Claude와 ChatGPT 같은 web client가 자연어로 호출하는 4층 구조다.

Source layer

DART API, DART web, KIND, 보조 source

Service layer

parser, mapping, fallback, law layer, scoring

Tool layer

company, meeting, data, action tools

Client layer

remote MCP connector / app surface

Contract

사용자는 source를 직접 보지 않고 tool contract를 본다. complexity는 service layer가 흡수한다.
