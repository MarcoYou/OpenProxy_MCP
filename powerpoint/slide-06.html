Slide 06

pre-meeting / DART notice parsing

shareholder_meeting_notice

주총 사전 분석의 핵심 source다. 소집공고에서 안건, 후보, 보수한도, 정관변경, 재무제표 관련 사실을 scope별로 구조화한다.

Role

사전 주총 소집공고를 parsing해 board / compensation / aoi_change 같은 scope로 분해한다.

Inputs

  • company or corp_code
  • year / meeting_type
  • scope = summary | board | compensation | aoi_change | prov_financials

Outputs

  • agenda hierarchy
  • candidate / compensation / amendment facts
  • notice metadata + evidence

Sources

  • DART list.json
  • DART document viewer HTML
  • request-local soup cache

Workflow

  • notice filing 탐색
  • HTML parsing + hierarchy extraction
  • scope별 payload 반환

Notes

최근 최적화로 request-local soup reuse를 넣었고 broad sample equality를 확인했다.
