dilutive_issuance

유상증자, CB, BW, 감자 등 희석성 이벤트를 묶어 dilution risk를 빠르게 보는 tool이다.

Role

희석 이벤트와 관련 terms를 한 곳에 모으는 issuance tab.

Inputs

  • company
  • lookback window
  • default summary scope

Outputs

  • issuance event summary
  • dilution-related terms
  • event type breakdown

Sources

  • DS005 issuance endpoints
  • event filings parsing
  • refixing / reduction rules

Workflow

  • 희석 이벤트 병렬 수집
  • event/terms 정규화
  • risk-oriented summary 반환

Notes

재무적 이벤트 해석과 proxy action 보조 fact source로 쓰기 좋다.
