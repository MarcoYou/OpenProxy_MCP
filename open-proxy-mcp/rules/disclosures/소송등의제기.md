# 소송등의 제기·신청·판결·결정

## 개요

회사가 당사자(원고/피고)인 소송, 가처분, 신청 등이 제기·판결·결정될 때 제출하는 의무 공시. 거버넌스 관점에서는 **경영권 분쟁의 시간순 신호**.

4종 세부 분류:
- 소송등의 제기 (소장 접수)
- 소송등의 신청 (가처분 등 임시조치 신청)
- 소송등의 판결
- 소송등의 결정

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=I (거래소공시) 또는 B (주요사항보고)
- **DART API**: 전용 구조화 API 없음. `list.json` + 제목 키워드 매칭
- **report_nm 패턴**:
  - `소송등의제기ㆍ신청`
  - `소송등의제기`
  - `소송등의신청`
  - `소송등의판결ㆍ결정`
  - `소송등의판결`
  - `소송등의결정`
- **의무 여부**: 의무
- **공시 주기**: 수시 (소장 수령/판결 수령 즉시)

## 키워드 (OPM 사용)

```python
_LITIGATION_KEYWORDS = (
    "소송등의제기",
    "소송등의신청",
    "소송등의판결",
    "소송등의결정",
    "경영권분쟁소송",
)
```

`경영권분쟁소송`은 별도 페이지 [[경영권분쟁소송]]으로 분리 관리.

## 전체 문서 구조 (트리)

표준 서식:

```
[회사명]/소송등의제기ㆍ신청/...
│
├── 1. 사건명                         (예: "주주총회 결의 무효 확인의 소", "회계장부열람 가처분")
├── 2. 원고 (Plaintiff)
│   ├── 성명 / 회사명
│   └── 발행회사와의 관계
├── 3. 피고 (Defendant)
│   ├── 성명 / 회사명
│   └── 발행회사와의 관계
├── 4. 청구취지                        (요구 사항)
├── 5. 청구원인                        (소송 사유)
├── 6. 관할 법원                       (예: "서울중앙지방법원")
├── 7. 사건번호                        (예: "2024가합12345")
├── 8. 사건 발생일자                    (소장 접수일)
├── 9. 소송가액                        (해당 시)
├── 10. 회사에 미치는 영향              (자유기재)
└── 11. 향후 대응 방안                  (자유기재)
```

## API 필드 vs 원문 대응

별도 구조화 API 없음. `list.json` 응답 기본 필드만:

| API 필드 | 의미 | 비고 |
|----------|------|------|
| `corp_code`, `corp_name` | 회사 | |
| `report_nm` | 공시 제목 | 사건 종류 + 사건명 일부 포함되는 경우 많음 (예: "소송등의제기ㆍ신청 (주주총회 결의 무효 확인의 소)") |
| `rcept_no`, `rcept_dt` | 접수번호/접수일 | |
| `flr_nm` | 제출인 (보통 회사) | |

**원문 상세**: 원고/피고/청구취지/사건번호는 모두 원문(get_document) 파싱 필요.

## OPM에서의 활용

- **`proxy_contest(scope="litigation")`**: 회사별 소송 timeline + 경영권분쟁 카테고리 분류
- **`screen_events(event_type="litigation")`**: 시장 전체 최근 소송 발생 기업 검색
- **`evidence(rcept_no=...)`**: 원문 viewer URL

## 거버넌스 분석 포인트

### 거버넌스 관련 소송 식별 (제목 키워드)

| 키워드 | 의미 | 거버넌스 시사점 |
|--------|------|----------------|
| **주주총회 결의 무효** / **취소** | 주총 적법성 분쟁 | 경영권 분쟁 핵심 |
| **이사 직무집행정지** | 이사 직무 정지 가처분 | 분쟁 가속 단계 |
| **회계장부 열람 등사** | 소수주주의 정보접근권 | 행동주의 / 분쟁 초기 |
| **주식발행금지** | 신주발행 가처분 | 우호 지분 형성 차단 |
| **임시주주총회 소집허가** | 소수주주 주총 소집 | 프록시 파이트 도구 |
| **주주명부 열람** | 의결권 권유 준비 | 위임장 권유 사전 단계 |
| **신주배정금지** | CB/유상증자 막기 | 희석 방어 |

### 회사 관련 일반 소송

- 거래처 분쟁, 손해배상 등은 거버넌스와 무관
- 다만 소송가액이 매출 5% 이상이면 재무위기 신호

### 원고가 회사 vs 피고가 회사

- **원고가 회사**: 회사가 적극 권리행사 (정상)
- **피고가 회사**: 외부 도전 받는 상태 — 경영권 분쟁 가능성 높음

### 판결·결정 단계

- **가처분 인용**: 분쟁 즉시 영향 (이사 직무정지, 주총 금지 등)
- **본안 판결**: 회사 승소/패소 결과. 경영권 향배 결정
- **판결 후 항소/상고**: 분쟁 장기화 신호
