## 개요

반기(6개월) 경과 후 45일 이내 제출하는 중간 보고서. 중간 재무제표와 반기 배당 정보 포함.

## 소스

- **제출처**: [[DART-OpenAPI]] (reprt_code=11012)
- **의무 여부**: 의무
- **공시 주기**: 정기 (반기, 사업연도 반기 종료 후 45일 이내)

## 주요 내용

- 중간 연결/별도 재무제표
- 반기 배당 현황 (해당 시)
- 주요 경영 사항 변동

## 전체 문서 구조 (트리)

```
반기보고서 (12월 결산법인 기준 → 6월말 제출)
├── 제1부 회사의 개요
├── 제2부 사업의 내용 (요약)
├── 제3부 재무에 관한 사항
│   ├── 연결 반기 재무제표
│   │   └── 반기 배당 정보 포함 가능 ← div_* 소스
│   └── 별도 반기 재무제표
└── 사업보고서 대비 간략화된 구조
```

API: DART reprt_code="11012" (반기보고서)
주의: 중간배당 정보가 여기 있을 수 있음 (분기배당 합산 시 주의)

## 파싱 주의사항

- `reprt_code="11012"`로 반기보고서 한정 조회
- 중간배당(반기배당) DPS가 연간 합산 대상인지 확인 필요 — 사업보고서 기말배당에 이미 포함된 경우 이중 계산 위험
- 사업보고서보다 구조가 간략하므로, 세부 재무 데이터는 사업보고서를 우선 참조

## OPM에서의 활용

- div_detail: reprt_code=11012로 반기 배당 DPS 조회
- [[분기배당]] 기업의 연간 DPS 합산 시 반기분 포함
- [[div-tool-rule]]에서 분기별 DPS 데이터 소스로 활용

## 관련

[[사업보고서]]의 중간 버전. [[분기보고서]]와 함께 분기별 배당 데이터 제공.
