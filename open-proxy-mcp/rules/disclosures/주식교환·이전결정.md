# 주식교환·이전 결정

## 개요

이사회에서 주식의 포괄적 교환 또는 포괄적 이전을 결정하면 제출하는 의무 공시. 지주회사 전환과 100% 자회사 편입의 정형 도구.
- **주식교환**: 두 회사 주주가 주식을 맞바꿔 한 회사가 다른 회사의 100% 모회사가 됨 (기존 모회사가 자회사를 완전 흡수)
- **주식이전**: 새로 설립되는 모회사에 주식을 이전 (지주회사 신설형)

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=B (주요사항보고)
- **DART API**: `stkExtrDecsn.json` (DS005 주요사항보고서)
- **report_nm 패턴**: `주요사항보고서(주식교환ㆍ이전결정)`, `[기재정정]주요사항보고서(주식교환ㆍ이전결정)`
- **의무 여부**: 의무
- **공시 주기**: 수시 (이사회 결의 익영업일까지)

## API 필드 vs 원문 대응

| `extr_sen` | 1. 종류 | 주식교환 / 주식이전 | 주식교환=기존 모회사 흡수, 주식이전=신설 지주회사 |

## OPM에서의 활용

- **`corporate_restructuring(scope="share_exchange")`**: 카드 형태로 종류/비율/대상회사/일정/매수청구권 노출.
- **`corporate_restructuring(scope="summary")`**: 4종 timeline에서 대상회사명 + 교환비율 표시.

## 거버넌스 분석 포인트

1. **자회사 100% 편입**: `extr_tgcmp_rl_cmpn=자회사` + 비율 협상력 검토. 자회사 소수주주 보호.
2. **소규모 교환 (`extr_stn=소규모`)**: 주총 생략 → 소수주주 의결권 박탈. 매수청구권만 보호 수단.
3. **상장 → 비상장 전환**: 자회사가 상장사면 100% 편입 후 상장폐지. 매수청구권 가격이 핵심.
4. **외부평가 미해당**: 상장법인간 교환은 시장가 기준이라 외부평가 의무 없음. 이 경우 비율 산정의 객관성 별도 검토 필요.
5. **풋옵션 약정**: `popt_ctr_atn=예` → 우호 지분에 일방적 매도 권리 부여.
6. **지주회사 전환 패턴**: 주식이전 (`extr_sen=주식이전`)은 신설 지주회사 설립 → 거버넌스 구조 근본 재편.
