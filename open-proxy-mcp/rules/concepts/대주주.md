# 대주주

## 정의 (자본시장법 §9 ②, 소득세법 §94)

상장법인의 발행주식 또는 출자지분의 일정 비율 이상을 보유하거나, 일정 시가총액 이상을 보유하는 주주.

- **상장사 대주주**: 본인 + 특수관계인 합산 1% 이상 보유 또는 시가총액 10억원 이상 (소득세법 양도소득 과세 기준)
- **금투법 대주주**: 회사 임원 + 5% 이상 보유자 (자본시장법 §9 ②)

## [[최대주주]] · 5% 보유자와의 차이

```
[[최대주주]] = 본인+특수관계인 합산 최다 보유 (단일 그룹)
대주주     = 본인+특수관계인 합산 1%+ 또는 시총 10억+ (다수 가능)
5% 보유자   = 본인+특별관계자 합산 5%+ (5일 내 신고 의무)
```

- [[최대주주]]는 항상 대주주에 포함되지만, 모든 대주주가 최대주주는 아님
- 5% 보유자는 대주주의 부분집합
- [[동일인]]은 [[최대주주]] 그룹의 정점(재벌만 해당)

## 의무

- **양도소득 과세**: 양도세 trigger
- **단기매매차익 반환**: 임원·5% 이상 주주는 6개월 내 매수·매도(또는 매도·매수) 시 차익 반환 의무 (자본시장법 §172)
- **신고 의무**: 5% 이상이면 [[5%-대량보유]] 보고 + 임원·10% 보유자는 elestock(소유보고)

## DART에서의 표기

- 별도의 단일 API 없음. 다음 source의 조합으로 도출:
  - `hyslrSttus` (최대주주 + 특수관계인) — 본인+특관 합산
  - `majorstock` (5% 보고) — 5% 이상 외부 주주
  - `elestock` (임원·주요주주 소유) — 임원+10% 보유자
  - 사업보고서 본문 (대주주 명단 — 회사별 표 형식 다양)

## OPM 사용

- `ownership_structure` tool의 `major_holders` 필드 = hyslrSttus 본인+특관
- `blocks` 필드 = 5% 외부 보유자
- 1% 미만 일반 대주주는 사업보고서 본문 OCR/XBRL 파싱 필요 (현재 미지원)

## 거버넌스 분석에서의 의미

- 1% 이상 대주주의 의결권 행사 패턴이 [[프록시-파이트]] 결과를 좌우
- 외국인 패시브 펀드(BlackRock, Vanguard 등)가 5% 이상 보유 시 ESG 의결권 행사 영향력 큼
