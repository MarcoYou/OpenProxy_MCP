# 네이버 금융

## 개요

네이버 금융(finance.naver.com). 한국 주식 시세, 시가총액 등을 제공하는 포털. [[DART-OpenAPI]]와 [[KRX-KIND]]에서 직접 제공하지 않는 종가 데이터의 보조 소스.

## OPM에서의 활용

### 시세 데이터
- [[배당수익률]] 자체 계산 시 종가 소스 (DART [[시가배당률]] 없을 때 fallback)
- 배당기준일 또는 직전 거래일 종가

### 시가총액
- market_cap.json: [[OpenProxy-AI]] 프론트엔드에서 기업 정렬용
- 수시 갱신

### 뉴스 검색 API
- `openapi.naver.com/v1/search/news.json`
- news_check tool: 이사/감사 후보자 부정 뉴스 검색
- 경력 기반 멀티 검색 + 부정 키워드 33개 필터링
- 주요 일간지 11개 우선 표시 (한경/매경/조선/중앙/동아/서울경제/한국일보/파이낸셜/이데일리/머니투데이/연합)
- 일일 25,000 호출 한도

### 기타
- 192개 기업 로고 (AlphaSquare CDN 경유)
